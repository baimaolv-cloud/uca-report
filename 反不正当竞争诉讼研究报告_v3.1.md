# 第一章 反不正当竞争法律体系概述

## 1.1 立法历程与修法背景

### 1.1.1 立法背景与总体脉络

《中华人民共和国反不正当竞争法》（以下简称"《反不正当竞争法》"或"反法"）是中国规范市场交易秩序、保护公平竞争的基础性法律。自1993年颁布以来，该法经历了2017年全面修订、2019年局部修正、2025年重大修订三次重要变革，从最初33条扩展至41条，规制范围从传统不正当竞争行为延伸至平台经济、数据竞争等新兴领域，体现了中国市场经济法治建设从"粗放立规"向"精准治理"的转型。

反不正当竞争法的立法演变可划分为四个阶段：

**表1-1 立法演变四阶段**

| 阶段 | 时间节点 | 立法机关 | 条文数量 | 核心变化 |
|------|---------|---------|---------|---------|
| 创制期 | 1993年9月2日 | 第八届全国人大常委会第三次会议 | 33条 | 首次立法，列举十一种不正当竞争行为 |
| 体系重构期 | 2017年11月4日 | 第十二届全国人大常委会第三次会议 | 33条（结构调整） | 新增互联网专条，完善商业秘密保护，混淆行为认定标准调整 |
| 微调期 | 2019年4月23日 | 第十三届全国人大常委会第十次会议 | 33条（局部修正） | 技术性、文字性调整 |
| 深化拓展期 | 2025年6月27日 | 第十四届全国人大常委会第十六次会议 | 41条 | 新增平台经济规制、数据保护专款、滥用优势地位、域外适用、商业秘密举证倒置等 |

### 1.1.2 1993年原版：市场经济初期的奠基性立法

1993年《反不正当竞争法》是在中国确立社会主义市场经济体制的背景下诞生的。该法作为中国第一部系统规范市场不正当竞争行为的法律，具有里程碑意义。其主要特征包括：

1. **行为列举模式**：采用"定义+列举"的立法技术，在第二条一般条款之后，具体列举了十一种不正当竞争行为，涵盖混淆、商业贿赂、虚假宣传、侵犯商业秘密、违规有奖销售、商业诋毁等传统类型。
2. **行政主导色彩**：执法主体为县级以上监督检查部门（实践中主要为工商行政管理部门），行政处罚力度相对有限。
3. **时代局限**：该法制定于互联网经济尚未兴起的年代，未涉及网络不正当竞争、数据竞争等问题；商业秘密保护条款较为粗疏，对电子侵入等新型侵权手段缺乏预见。

### 1.1.3 2017年修订：回应互联网经济的首次重大转型

2017年修订是反法颁布二十四年后的首次全面修订，核心变化体现在以下方面：

1. **新增互联网不正当竞争条款**：新增第12条（2017年版），首次将网络领域的不正当竞争行为纳入规制范围，列举了强制跳转、误导卸载、恶意不兼容等行为类型，回应了互联网行业快速发展的法律需求。
2. **商业秘密保护完善**：明确将电子侵入纳入侵犯商业秘密的手段范围，扩大了侵权行为的认定口径。
3. **混淆行为认定标准调整**：将原"知名商品"的表述修改为"有一定影响"，降低了权利人的举证门槛，使保护范围从"知名"这一较高标准扩展至"有一定影响"这一更广的范畴。
4. **违法行为处罚力度加重**：混淆行为罚款上限从原有的1倍提升至5倍，商业秘密侵权等行为的行政处罚幅度也相应提高。

### 1.1.4 2019年修正：技术性微调

2019年4月23日的修正内容有限，主要为文字性和技术性调整，未涉及制度性变革。此次修正的意义在于维持法律的时效性，为后续更大幅度的修订预留制度空间。

### 1.1.5 2025年修订：平台经济与数据时代的制度跨越

2025年6月27日，第十四届全国人大常委会第十六次会议对反法进行第二次修订，于2025年10月15日起施行。此次修订是反法历史上幅度最大的一次，条文从33条增加至41条，新增8条，修改多条，核心亮点包括：

**（1）平台经济全面规制**

新增《反不正当竞争法》第十四条（平台强制低价），禁止平台经营者利用技术手段、平台规则等方式，强制平台内经营者以低于成本的价格销售商品。新增《反不正当竞争法》第十五条（滥用优势地位），规制具有市场优势地位的经营者滥用其地位实施不正当竞争行为。上述两条共同构成了平台经济领域反不正当竞争的制度框架，与《电子商务法》第三十五条规定形成规制交叉。

**（2）数据保护专款入法**

2025年修订在《反不正当竞争法》第十三条（互联网专条）中新增数据保护专款，明确禁止以不正当方式获取、使用其他经营者合法持有的数据。这是中国首次在反法层面确立数据的专门保护规则，孔祥俊教授在《反不正当竞争法释论与适用》（2025年）中称其为"数据保护立法的重要突破"，标志着数据权益保护从司法实践层面的个案裁判，上升至立法层面的制度性确认。

**（3）商业秘密保护强化**

新增《反不正当竞争法》第三十九条，确立商业秘密侵权举证责任移转规则，显著降低了权利人的举证负担。结合既有惩罚性赔偿制度（故意侵犯商业秘密且情节严重的，可在实际损失或侵权获利的1~5倍范围内确定赔偿数额），商业秘密保护的威慑力大幅提升。

**（4）混淆行为扩展至新领域**

2025年修订将混淆行为的规制范围扩展至搜索关键词和新媒体账号等新型标识使用场景，体现了法律对数字营销领域新问题的关注。

**（5）域外适用效力**

新增《反不正当竞争法》第四十条，确立反法的域外适用效力，对在中华人民共和国境外实施的不正当竞争行为，扰乱境内市场秩序或者损害境内经营者合法权益的，依照本法处理。这一条款为跨境不正当竞争纠纷的解决提供了法律依据。

**（6）约谈制度与平台义务**

新增《反不正当竞争法》第十八条（约谈制度），赋予执法机关对平台经营者进行约谈的权力。新增《反不正当竞争法》第二十一条，要求平台经营者建立健全投诉处置机制，强化平台在竞争治理中的主体责任。

### 1.1.6 2025年修订与2017年修订的比较

为更清晰地理解2025年修订的突破性意义，有必要将其与2017年修订进行系统性比较：

**表1-2 2017年修订与2025年修订比较**

| 比较维度 | 2017年修订 | 2025年修订 |
|---------|-----------|------------|
| **修订背景** | 互联网经济兴起，回应网络不正当竞争问题 | 数字经济深化，数据成为核心竞争资源，平台经济问题凸显 |
| **条文数量** | 33条（未变，结构调整为主） | 41条（新增8条，修改多条） |
| **新增条款** | 互联网专条（第12条） | 平台强制低价（第14条）、滥用优势地位（第15条）、商业秘密举证责任移转（第39条）、域外适用（第40条） |
| **互联网条款** | 原则性规定，列举典型行为 | 原则性规定+数据保护专款+滥用平台规则禁止 |
| **数据规制** | 未明确规定 | 首次在立法层面明确数据保护规则 |
| **责任主体** | 以单位为主 | 单位+个人双重责任（商业贿赂等条款） |
| **处罚力度** | 罚款上限普遍提高；商业贿赂最高300万元 | 罚款上限进一步提高；商业贿赂最高500万元 |
| **治理模式** | 事后惩戒为主 | 事前预防（约谈制度）+事中监管（平台义务）+事后惩戒 |
| **域外效力** | 未明确规定 | 明确域外适用条款 |

从上述比较可以看出，2017年修订是"回应型"修法，主要解决互联网兴起带来的表层问题；2025年修订则是"前瞻型"修法，不仅回应既有问题，更着眼于数字经济深化发展可能带来的新型竞争形态，展现了立法者对市场发展趋势的前瞻性判断。

### 1.1.7 一般条款的演变：由封闭、开放到谦抑

反法第二条一般条款的适用范围，在历次修法的演进中呈现出"封闭→开放→谦抑"的辩证轨迹，这一理论脉络由孔祥俊教授在《反不正当竞争法范式论》中系统阐述。

**（1）封闭期（1993年原版）**

1993年原版反法虽设有第二条一般条款，但司法实践对其适用长期持谨慎态度。由于该法已列举十一种具体行为类型，法院倾向于认为一般条款仅具宣示意义，不应独立于具体列举条款之外创设新的行为类型。这一时期的司法态度呈现出明显的"封闭"特征——一般条款的适用被严格限制在列举行为之外的空间极为有限。

**（2）开放期（2017年修订后）**

2017年修订删除了原法中部分已不合时宜的行为条款（如公用企业强制交易、搭售等），同时新增互联网专条，但新旧交替之间仍留下大量法律空白。伴随互联网经济的爆发式增长，新型竞争争议层出不穷——数据抓取、流量劫持、恶意屏蔽等行为亟需法律回应。在此背景下，法院对第二条一般条款的适用逐渐走向"开放"，大量新型不正当竞争行为通过一般条款获得规制。这一时期，一般条款成为司法应对新型竞争问题的"利器"，但也在一定程度上带来了适用标准不统一、裁判尺度不一的问题。

**（3）谦抑期（2025年修订后）**

经历了开放期的实践检验后，理论界和实务界逐渐认识到，一般条款的过度扩张可能侵蚀知识产权专门法的制度边界，将本属于公有领域的自由竞争行为不当纳入反法规制范围。孔祥俊教授在《反不正当竞争法释论与适用》（2025年）中明确指出，一般条款的适用应"从严把握"，奉行谦抑立场——反不正当竞争法不应成为扩张知识产权保护的"后门"，在专门法上不属于侵权的行为，属于公有领域的自由行为，不应再由反法制止。2025年修订在扩充具体行为类型（平台强制低价、滥用优势地位等）的同时，客观上压缩了一般条款的适用空间，体现了立法者对一般条款谦抑适用的制度回应。

### 1.1.8 从"竞争关系"到"竞争行为本位"的理论转向

传统反不正当竞争法理论长期将"竞争关系"视为不正当竞争行为的构成要件和反法的适用前提，即原告须证明与被告之间存在同业竞争关系，方可主张反法保护。这一理论框架在工业经济时代尚可适用，但在互联网经济跨界竞争日趋普遍的背景下，竞争关系的认定面临严峻挑战——平台企业之间往往并不处于同一行业，却存在激烈的竞争关系。

孔祥俊教授在《反不正当竞争法释论与适用》（2025年）中系统论证了"竞争行为本位论"，核心观点包括：

1. **竞争关系非反法适用前提**：竞争关系既非不正当竞争行为的构成要件，更非反法的适用条件。不正当竞争的本质在于"行为的不正当性"，而非行为主体之间是否存在竞争关系。
2. **应从"竞争行为"而非"竞争关系"角度理解不正当竞争**：判断某一行为是否构成不正当竞争，应聚焦于行为本身的性质——是否违反诚实信用原则和商业道德、是否扰乱市场竞争秩序，而非先判断行为主体之间是否存在竞争关系。
3. **竞争关系仅具程序性意义**：竞争关系在诉讼中的功能，主要是作为认定损害可能性的考量因素，进而确定原告的适格地位，而非行为定性的前提条件。原告只需证明与被诉行为人存在"可能的争夺交易机会、损害竞争优势关系"即可，无需证明同业竞争关系。
4. **契合国际趋势**：从"竞争关系"到"竞争行为本位"的转向，契合《巴黎公约》第十条之二关于不正当竞争规制的现代精神，也与德国UWG、世界知识产权组织（WIPO）模范法等国际立法趋势相一致。

这一理论转向对司法实践产生了深远影响。法释〔2022〕9号第二条将"其他经营者"界定为"与被诉行为人存在可能的争夺交易机会、损害竞争优势关系的经营者"，实际上已采纳了竞争行为本位论的核心理念——原告资格的认定不再以同业竞争关系为前提，而是以竞争行为的损害可能性为标准。

## 1.2 2025年修订的核心变化

### 1.2.1 总则充实：基本原则的深化

2025年修订对总则部分进行了重要充实，尤其是对第二条（基本原则）的修改，体现了立法者对数字经济时代竞争规则的系统性思考。

《反不正当竞争法》（2025修订）第二条规定了反法的基本原则和核心概念，其修订要点包括：

**（1）诚信原则的强化**

2025年修订进一步强调了诚信原则在反法中的核心地位。诚信原则是反法的灵魂，是判断竞争行为正当性的根本标准。孔祥俊教授在《反不正当竞争法释论与适用》（2025年）中指出，在数字经济时代，诚信原则的内涵有了新的扩展——不仅包括传统商业交往中的诚实守信，还包括对平台规则、数据权益、用户隐私等方面的尊重与保护。

**（2）"经营者"定义的完善**

2025年修订对"经营者"定义进行了完善，将平台经营者、利用网络提供服务的新型市场主体明确纳入经营者范畴。这一修改回应了实践中关于平台经营者是否属于反法规制对象的争议，为平台经济的规范发展提供了明确的法律依据。

**（3）"不正当竞争行为"定义的厘清**

2025年修订进一步厘清了"不正当竞争行为"的定义，明确其应具备以下要素：扰乱市场竞争秩序、损害其他经营者或者消费者合法权益。这一要素化定义不仅为司法裁判提供了更清晰的指引，也为经营者自我合规提供了明确的行为边界。

### 1.2.2 数据保护专款：首次入法的里程碑意义

2025年修订在第十三条（互联网专条）中新增数据保护专款（第二款），首次在立法层面确立数据保护规则，是本次修法最大的制度亮点之一。该专款明确了禁止以不正当方式获取、使用其他经营者合法持有的数据，是中国数据保护立法的重要突破。

**（1）条文内容**

《反不正当竞争法》（2025修订）第十三条第二款规定：

> "经营者不得以欺诈、胁迫、避开或者破坏技术管理措施等不正当方式，获取、使用其他经营者合法持有的数据，损害其他经营者的合法权益，扰乱市场竞争秩序。"

**（2）构成要件**

数据保护专款的适用须同时满足以下三要素：

**表1-3 数据保护专款构成要件**

| 要素 | 内容 | 说明 |
|------|------|------|
| 不正当手段 | 欺诈、胁迫、避开或破坏技术管理措施等 | 强调手段的"不正当性"，排除了合法途径获取数据的行为 |
| 受保护数据的适格性 | 须为"合法持有的数据" | 数据须为其他经营者合法持有，公开可获取的数据不属此列 |
| 损害后果 | 损害其他经营者的合法权益，扰乱市场竞争秩序 | 须造成实质性损害 |

**（3）豁免情形**

孔祥俊教授在《反不正当竞争法释论与适用》（2025年）中指出，数据保护专款设定了以下豁免情形，为正当的数据流通和利用预留了制度空间：

1. 可从公开渠道自由获取的数据——如果数据可以通过公开渠道自由获取，则不受该条款保护；
2. 使用数量不足以造成实质性损害——偶发的、小规模的引用不构成违法；
3. 不与权利人正常经营冲突——善意、合理的利用行为可获豁免；
4. 以诚信方式使用的其他情形——兜底条款，保留法官个案裁量的空间。

**（4）里程碑意义**

孔祥俊教授将数据保护专款的入法称为"数据保护立法的重要突破"，其意义体现在以下层面：

1. **从司法裁判到立法规制**：此前，数据抓取行为的正当性判断主要依赖司法裁判，裁判标准因案而异。数据保护专款的确立，为数据竞争纠纷提供了明确的法律标准，增强了法律的可预期性。
2. **从行为规制到权益确认**：数据保护专款不仅规制不正当的数据获取行为，更从立法层面确认了数据的财产属性和权益归属，有助于数据要素市场的规范化发展。
3. **从补充保护到专门保护**：此前，数据权益主要通过反法一般条款获得补充性保护。数据保护专款的确立，标志着数据权益获得了专门的法律保护地位。

### 1.2.3 混淆行为扩展：涵盖新媒体标识与搜索关键词

2025年修订对《反不正当竞争法》第七条（混淆行为）进行了重要扩展，将混淆行为的规制范围延伸至新媒体账号和搜索关键词等数字标识领域。

**（1）扩展背景**

在传统商业模式下，混淆行为的对象主要是商品名称、商品包装、商品装潢等实体标识。随着数字经济的发展，网站域名、新媒体账号名称、搜索关键词等数字标识日益成为经营者识别来源的重要载体，也成为混淆行为的新载体。2017年修订虽已将混淆行为的对象从"知名商品特有的名称、包装、装潢"扩展至"有一定影响的标识"，但未明确涵盖新媒体账号和搜索关键词。2025年修订弥补了这一立法空白，将数字标识正式纳入混淆行为的规制范围。

**（2）2025年修订的三个扩展方向**

2025年修订进一步明确，混淆行为的对象包括：

1. **新媒体账号名称**：包括微信公众号、微博账号、抖音账号、小红书账号等平台账号名称。实践中，假冒他人公众号或微博账号吸粉导流的侵权行为此前只能通过一般条款规制，2025年修订后有了明确的请求权基础。
2. **搜索关键词**：包括搜索引擎关键词、电商平台搜索关键词等，用于引导用户流量。搜索关键词混淆典型表现是将他人注册商标或商号设置为自己的搜索关键词，导致用户在搜索时优先看到侵权人的信息。
3. **其他数字标识**：包括网站名称、应用程序名称等具有识别来源功能的数字标识。随着移动互联网的普及，应用程序名称和网站名称已成为消费者识别商品和服务来源的重要依据。

**（3）实务要点**

1. **显著性要求**：新媒体账号名称和搜索关键词须具有"有一定影响"的显著性，方可获得反法保护。"有一定影响"的认定需综合考量标识的使用时间、宣传范围、地域范围等因素。
2. **混淆可能性的判断**：需综合考量标识的相似度、消费群体的认知习惯、被告的主观意图等因素。在数字营销场景下，混淆可能性的判断还需考虑搜索算法的特性、关键词匹配规则等技术因素。
3. **帮助行为的规制**：为他人实施混淆行为提供帮助的，如为混淆行为人提供技术接口或平台服务，同样可能构成不正当竞争。

### 1.2.4 商业贿赂双向禁止：责任到人的制度创新

2025年修订对《反不正当竞争法》第八条（商业贿赂）进行了重要修改，建立了双向禁止的规制机制，并强化了个人责任的追究。这一修改回应了实践中商业贿赂行为日益隐蔽化、多元化的趋势，体现了立法者对商业贿赂行为零容忍的鲜明态度。

**（1）双向禁止机制的完整含义**

《反不正当竞争法》（2025修订）第八条确立了商业贿赂的双向禁止机制，即同时禁止行贿行为和受贿行为：

**表1-4 商业贿赂双向禁止机制**

| 禁止方向 | 受禁止行为 | 行为要件 | 处罚标准 |
|---------|-----------|---------|---------|
| 禁止行贿 | 经营者为谋取交易机会或者竞争优势，采用财物或者其他手段贿赂相关单位或者个人 | 1. 主体为经营者；2. 主观上为谋取交易机会或竞争优势；3. 手段为财物或其他利益；4. 贿赂对象为相关单位或个人 | 单位：10万元以上100万元以下罚款；情节严重的100万元以上500万元以下罚款 |
| 禁止受贿 | 相关单位或者个人利用职权或者影响力，索取、收受经营者的财物或者其他利益 | 1. 主体为相关单位或个人；2. 利用职权或影响力；3. 主动索取或被动收受 | 单位：最高200万元罚款；个人：50万元以下罚款 |

双向禁止机制的建立填补了此前仅禁止行贿、疏于规制受贿的制度漏洞，有助于从源头上遏制商业贿赂行为。

**（2）责任到人的制度创新**

2025年修订的另一重要创新是确立了商业贿赂的个人责任，标志着反法从单纯规制单位行为向同时追究个人责任的转型：

1. **个人行贿的独立处罚**：个人实施商业贿赂行为的，可处100万元以下罚款。这意味着即便不依托单位，个人实施行贿行为同样会受到行政处罚。
2. **个人受贿的独立处罚**：个人利用职权或者影响力收受贿赂的，可处50万元以下罚款。典型场景如采购人员、医疗从业人员、教育领域人员利用职务便利收受回扣。
3. **直接责任人员的责任**：单位实施商业贿赂行为的，对直接负责的主管人员和其他直接责任人员可并处处罚。这一规定强化了企业内部合规管理的责任主体。

**（3）实务要点**

1. **贿赂的认定**：财物以外的"其他手段"包括旅游招待、会员服务、股权激励、咨询顾问费等非财产性利益。实务中判断是否属于"其他手段"，应综合考量利益的客观价值、给予方式、双方关系等因素。
2. **交易机会或竞争优势的关联性**：贿赂须与谋取交易机会或竞争优势存在因果关系。单纯的好意施惠（如节日礼金）若与业务往来无直接关联，一般不构成商业贿赂。
3. **主观故意的认定**：商业贿赂须以主观故意为要件，过失给付不当利益不构成商业贿赂。但若事后知情且不予退还，仍可能构成商业贿赂。

### 1.2.5 修法对实务的影响

2025年修订对反不正当竞争实务产生了深远影响，主要体现在以下几个方面：

**（1）诉讼策略调整**

1. **数据权益诉讼的举证重点**：在数据抓取纠纷中，原告须证明：（1）其对系争数据享有合法权益；（2）被告采用了不正当手段获取数据（如欺诈、胁迫、破坏技术措施等）；（3）被告的行为造成了实质性损害。举证责任移转规则在商业秘密案件中的适用，减轻了原告的举证负担，但数据权益诉讼目前尚无此规则，原告仍需承担较重的举证责任。
2. **混淆行为案件的扩展**：诉讼策略可考虑将新媒体账号名称、搜索关键词纳入混淆行为的诉请范围，扩大权利保护范围。新媒体账号混淆案件的管辖法院可能因平台所在地而异，需提前评估诉讼成本。
3. **域外侵权诉讼的增加**：第四十条域外适用条款的确立，为跨境不正当竞争纠纷的管辖提供了明确依据，预计将引发更多涉及境外经营者的诉讼。原告需关注证据跨境调取的可行性和域外送达的效率问题。

**（2）企业合规建议**

1. **数据合规体系建设**：企业应建立完善的数据合规体系，明确数据的分类分级、访问权限、使用审批等管理流程，确保数据获取和使用的合法性。建议参照《数据安全法》《个人信息保护法》的要求，建立数据资产目录和分类分级管理制度。
2. **平台规则审查**：平台经营者应审查平台规则的合法性，避免因滥用平台规则而触发反法第十三条第二款（滥用平台规则）和第十五条（滥用优势地位）的规制。平台规则的制定应遵循公平、透明原则，不得设置不合理条件限制平台内经营者。
3. **商业贿赂风险防范**：企业应建立商业贿赂风险防范机制，对员工、经销商、合作方等的行为进行合规管理。建议建立礼品招待登记制度、第三方尽职调查程序、举报热线等内控措施。

**（3）执法趋势预测**

1. **数据执法案件增加**：数据保护专款的确立将激发执法机关对数据不正当竞争行为的查处力度，预计数据抓取纠纷将成为执法热点。市场监管部门可能出台数据不正当竞争的执法指南或典型案例。
2. **平台经济执法加强**：第十四条（平台强制低价）和第十五条（滥用优势地位）的新增，为平台经济领域的专项执法提供了明确依据。预计执法机关将开展平台经济专项执法行动，重点关注"二选一""独家合作"等行为。
3. **跨部门执法协作**：约谈制度和平台义务的建立，将推动市场监管部门与网信部门、工信部门之间的执法协作。在涉及数据安全、算法推荐等复杂案件中，多部门联合执法的情形可能增多。

## 1.3 总则条款逐条解读

### 1.3.1 第一条：立法目的

《反不正当竞争法》（2025修订）第一条规定：

> "为促进社会主义市场经济健康发展，鼓励和保护公平竞争，预防和制止不正当竞争行为，保护经营者和消费者的合法权益，制定本法。"

本条规定了反法的立法目的，包括四个层次：

1. **促进社会主义市场经济健康发展**——体现了反法的宏观调控功能。反不正当竞争法不仅是微观层面的行为规制法律，更是宏观层面的市场经济秩序维护法律，其根本目标在于维护健康有序的市场经济运行环境。
2. **鼓励和保护公平竞争**——体现了反法对竞争自由和公平秩序的双重保护。"鼓励竞争"强调竞争自由，"保护公平竞争"强调竞争秩序，两者相互支撑、不可偏废。
3. **预防和制止不正当竞争行为**——体现了反法从"事后救济"向"事前预防"的治理转型。2025年修订新增的约谈制度和平台义务条款，进一步强化了预防功能。
4. **保护经营者和消费者的合法权益**——体现了反法的"三元利益叠加"保护框架。经营者和消费者是市场竞争的两大核心参与者，反法的保护范围覆盖两者。

"三元利益叠加"分析框架可图示如下：

```
              +------------------+
              |    公共利益       | <-- 更高位阶
              |   （市场秩序）    |
              +--------+---------+
                       |
          +------------+-------------+
          |                          |
    +-----+-----+            +-----+-----+
    |  竞争者利益  |            |  消费者权益  |
    |  （经营者）  |            |  （消费者）  |
    +-----------+            +-----------+
```

### 1.3.2 第二条：基本原则与核心概念

《反不正当竞争法》（2025修订）第二条规定：

> "经营者在生产经营活动中，应当遵循自愿、平等、公平、诚信的原则，遵守法律和商业道德。
>
> 本法所称经营者，是指从事商品生产、经营或者提供服务（以下统称商品）的自然人、法人和非法人组织。
>
> 本法所称不正当竞争行为，是指经营者在生产经营活动中，违反本法规定，扰乱市场竞争秩序，损害其他经营者或者消费者的合法权益的行为。
>
> 在中华人民共和国境外从事不正当竞争行为，扰乱境内市场秩序或者损害境内经营者或者消费者合法权益的，依照本法处理。"

本条是反法的核心条款，规定了四项核心内容：

1. **基本原则**：自愿、平等、公平、诚信四原则构成反法的基本原则体系，其中诚信原则是核心，是判断竞争行为正当性的根本标准。
2. **经营者定义**：将经营者界定为从事商品生产、经营或者提供服务的自然人、法人和非法人组织，明确了反法的适用主体范围。
3. **不正当竞争行为定义**：将不正当竞争行为界定为扰乱市场竞争秩序、损害其他经营者或者消费者合法权益的行为，确立了行为认定的三要素框架。
4. **域外适用效力**：明确规定域外适用条款，对境外不正当竞争行为造成境内损害的，中国法院具有管辖权。

### 1.3.3 第三条：党的领导与公平竞争审查

《反不正当竞争法》（2025修订）第三条规定：

> "从事生产经营活动，应当遵守法律，自觉接受政府和公众的监督。
>
> 各级人民政府应当采取措施，制止不正当竞争行为，为公平竞争创造良好的环境和条件。
>
> 国务院建立反不正当竞争工作协调机制，研究决定反不正当竞争工作中的重大问题，统筹协调重大反不正当竞争行政执法活动。"

本条确立了党的领导地位和公平竞争审查制度，体现了反法的政治站位和制度保障。国务院反不正当竞争工作协调机制的建立，为跨部门执法协作提供了制度平台，有助于解决实践中部门职能交叉、执法标准不一等问题。

### 1.3.4 第四条：政府职责

《反不正当竞争法》（2025修订）第四条规定：

> "各级人民政府应当采取措施，制止不正当竞争行为，为公平竞争创造良好的环境和条件。"

本条明确了各级人民政府在反不正当竞争工作中的职责定位——不仅是执法者，更是公平竞争环境的创造者和维护者。这一规定要求政府在产业政策、招投标、政府采购等领域自觉维护公平竞争秩序，不得实施或纵容地方保护、市场分割等破坏竞争的行为。

### 1.3.5 第五条：执法主体

《反不正当竞争法》（2025修订）第五条规定：

> "监督检查部门依照本法规定，负责对不正当竞争行为进行监督检查。"

本条明确了市场监管部门的主体执法地位。根据现行体制，县级以上人民政府市场监督管理部门是反不正当竞争法的主要执法机关，法律、行政法规规定由其他部门查处的，依照其规定。

### 1.3.6 第六条：社会监督

《反不正当竞争法》（2025修订）第六条规定：

> "国家鼓励、支持和保护一切组织和个人对不正当竞争行为进行社会监督。
>
> 行业组织应当加强行业自律，引导、规范会员依法竞争，维护市场竞争秩序。"

本条确立了社会监督和行业自律两大机制，体现了多元共治的治理理念。社会监督包括舆论监督、举报投诉等方式；行业自律则通过行业协会制定行规行约、建立诚信档案等方式实现。这两种机制与行政执法形成互补，共同构建反不正当竞争的社会共治格局。

## 1.4 本章小结

本章系统梳理了中国《反不正当竞争法》的立法沿革与制度框架，重点解读了2025年修订的核心变化。2025年修订是反法历史上幅度最大的一次制度变革，主要体现在以下方面：

1. **数据保护专款入法**：首次在立法层面确立数据的专门保护规则，回应了大数据时代的新型竞争争议，为数据要素市场的规范化发展提供了法律依据。孔祥俊教授将其评价为"数据保护立法的重要突破"。

2. **总则条款充实**：基本原则和核心概念的界定更加清晰，域外适用效力得以明确，为司法裁判和合规管理提供了明确指引。

3. **混淆行为扩展**：将新媒体账号名称、搜索关键词等数字标识纳入混淆行为的规制范围，体现了法律对数字营销领域新问题的关注，与时俱进地回应了实践需求。

4. **商业贿赂双向禁止**：建立双向禁止的规制机制，确立责任到人的制度创新，强化了对商业贿赂行为的威慑力，填补了此前仅禁止行贿的制度漏洞。

5. **平台经济全面规制**：新增平台强制低价和滥用优势地位两条，为平台经济领域的专项执法提供了明确依据，与《电子商务法》形成规制互补。

6. **域外适用效力明确**：为跨境不正当竞争纠纷的解决提供了法律依据，标志着中国反法与国际接轨的重要进展。

从立法沿革看，反法经历了从"封闭"到"开放"再到"谦抑"的演变轨迹。孔祥俊教授在《反不正当竞争法释论与适用》（2025年）中指出，2025年修订体现了立法者对数字经济时代竞争规律认识的深化，标志着中国反不正当竞争法律制度从"回应型"向"前瞻型"的转型。这一转型不仅回应了当下的市场关切，更着眼于数字经济深化发展可能带来的新型竞争形态，展现了法律制度的韧性与前瞻性。

## 1.5 法释〔2026〕7号惩罚性赔偿司法解释解读

### 1.5.1 法规信息与核心影响

**法规信息**：《最高人民法院关于审理侵害知识产权民事纠纷案件适用惩罚性赔偿的解释》，法释〔2026〕7号，2026年4月7日由最高人民法院审判委员会第1972次会议通过，**2026年5月1日起施行**。共14条。同时废止法释〔2021〕4号旧惩罚性赔偿解释。

**废止衔接说明**：法释〔2026〕7号施行后，原《最高人民法院关于审理侵害知识产权民事案件适用惩罚性赔偿的解释》（法释〔2021〕4号）同时废止。新旧解释名称存在差异：旧解释名称为"民事案件"，新解释名称为"民事纠纷案件"，体现了适用范围的微调。对于2026年5月1日前已生效裁判的再审案件，不溯及适用新解释；对2026年5月1日后新受理的一审、二审案件，无论侵权行为发生时间，均可适用本解释。

**对反法诉讼的核心影响——第5条**：

> 原告针对被告故意实施**侵犯商业秘密以外**的不正当竞争行为请求惩罚性赔偿的，人民法院**不予支持**，但法律另有规定的除外。

该条明确了反法领域惩罚性赔偿的适用范围：**仅限于商业秘密侵权案件**，其他类型的不正当竞争行为（如混淆、虚假宣传、商业诋毁、数据不正当竞争等）不适用惩罚性赔偿。这一规则与反法第22条第3款（"经营者恶意实施侵犯商业秘密行为，情节严重的，可以按照实际损失数额的一倍以上五倍以下确定赔偿数额"）相对应。

### 1.5.2 条文结构概览

| 条文 | 主题 | 核心内容 |
|---|---|---|
| 第1条 | 适用前提 | 故意+情节严重→惩罚性赔偿 |
| 第2条 | 请求明确性 | 须提出明确赔偿数额、计算方法及事实理由 |
| 第3条 | 增加请求时机 | 一审辩论终结前可增加；二审可调解，调解不成不支持 |
| 第4条 | 另行起诉禁止 | 经释明未请求，另行起诉不予受理 |
| **第5条** | **反法适用限制** | **商业秘密以外的不正当竞争行为不适用惩罚性赔偿** |
| 第6条 | "故意"认定 | 8种情形（通知后继续、关联关系、和解后再犯、隐名逃避等） |
| 第7条 | "情节严重"认定 | 7种情形（重复侵权、拒不履行保全、伪造证据、以侵权为业等） |
| 第8条 | 计算基数 | 实际损失/违法所得/侵权获利；法定赔偿不得作为基数 |
| 第9条 | 利润计算方法 | 营业利润→销售利润（以侵权为业）→行业平均利润率 |
| 第10条 | 举证妨碍 | 拒不提供账簿→依原告主张确定基数 |
| 第11条 | 倍数确定 | 综合主观过错+情节严重程度，可非整数 |
| 第12条 | 赔偿上限 | 最高5倍基数，合理开支另行计算 |
| 第13条 | 行政罚款/刑事罚金折抵 | 已处罚款/罚金在确定倍数时应予考虑 |
| 第14条 | 施行与溯及力 | 2026年5月1日施行；再审案件不适用 |

### 1.5.3 "故意"认定的8种情形（第6条）

1. 经有效通知后仍继续侵权；
2. 被告或其法定代表人/管理人是原告或利害关系人的法定代表人/管理人/实际控制人；
3. 存在劳动、劳务、合作、许可、经销、代理等关系且基于该关系接触过知识产权；
4. 有业务往来或合同磋商关系且基于该关系接触过知识产权；
5. 实施盗版、假冒注册商标、假冒专利行为；
6. **和解后再次实施相同或类似侵权行为**（新增）；
7. **通过设立关联公司、变更法定代表人/控股股东、隐名设立公司等掩盖实际控制关系，或签订免责协议逃避责任**（新增）；
8. 其他情形。

### 1.5.4 "情节严重"认定的7种情形（第7条）

1. 因侵权被行政处罚或裁判后再次实施相同侵权；
2. 无正当理由拒不履行保全裁定；
3. 伪造、毁坏或隐匿侵权证据；
4. 以侵权行为为主营业务或以侵权获利为主要利润来源（"以侵害知识产权为业"）；
5. 侵权获利巨大或导致权利人商誉、市场份额严重受损；
6. 侵权行为危害或可能危害国家利益、社会公共利益；
7. 其他情形。

### 1.5.5 计算基数与倍数确定规则

**计算基数规则（第8-10条）**：
- 基数=实际损失/违法所得/侵权获利，不包括合理开支；
- 三者均难以计算时，参照许可使用费确定；
- **法定赔偿数额不能作为惩罚性赔偿的计算基数**；
- 以侵权获利为基数时：一般参照营业利润；以侵权为业的参照销售利润；利润率无法确定的参照行业平均利润率或权利人利润率；
- 举证妨碍：被告拒不提供账簿资料的，法院可依原告主张确定基数。

**倍数确定规则（第11-13条）**：
- 综合考虑主观过错程度+情节严重程度，倍数可非整数；
- 赔偿总额上限=基数×5倍，合理开支另行计算；
- 同一侵权行为已被处以罚款/罚金且执行完毕的，确定倍数时应予考虑（过罚相当原则）。

### 1.5.6 溯及力规则（第14条）

- 第14条规定本解释自2026年5月1日起施行，再审案件不适用；
- 对2026年5月1日前已生效裁判的再审案件，不溯及适用本解释；
- 对2026年5月1日后新受理的一审、二审案件，无论侵权行为发生时间，均可适用本解释；
- 已于2026年5月1日前审结、当事人未申请再审的案件，不因本解释施行而当然进入再审。

**标杆案例对照——吉利诉威马案（6.4亿元案）**：

2024年6月14日最高人民法院（2023）最高法知民终1590号判决创中国知识产权侵权诉讼判赔数额历史最高纪录（约6.4亿元），其裁判规则与法释〔2026〕7号高度契合，是商业秘密领域惩罚性赔偿适用的标杆案例。该案适用2倍惩罚性赔偿，以"威马EX5整车利润×技术贡献率8%"为基数，合理开支1.4亿元另行计算；并创新判令未按期停止侵害的，按每日100万元计付迟延履行金。该案是法释〔2026〕7号第5—13条规则的最佳实务注脚，详见本报告3.3.5节。

## 1.6 司法解释适用中的待定问题

### 1.6.1 法释〔2022〕9号与2025年修订版的衔接

法释〔2022〕9号系基于2019年修正版反不正当竞争法制定。2025年修订版新增的第13条（数据专款）、第14条（平台强制低价）、第15条（滥用优势地位）、第39条（举证责任移转）、第40条（域外适用）等条款，法释〔2022〕9号均未涉及。最高人民法院公开表示正在加快推进相关司法解释的制定工作，但截至2026年6月20日尚未发布反不正当竞争法专项司法解释。实务中，对于新条文缺乏司法解释指引的问题，法院参照片面立法目的和学术理论进行裁判。

**注意**：法释〔2026〕7号系惩罚性赔偿专项司法解释，非反不正当竞争法专项司法解释。该解释第5条明确限定了反法领域惩罚性赔偿的适用范围（仅限商业秘密侵权），对反法诉讼具有重大影响，但不解决第13-15条、第39-40条等新条文的适用问题。

### 1.6.2 "有一定影响"在网络标识领域的适用

搜索关键词、新媒体账号等新型标识是否适用"有一定影响"的认定标准，仍需司法解释进一步明确。2025年修订虽已将搜索关键词和新媒体账号名称纳入第七条规制范围，但"有一定影响"在数字标识领域的具体认定因素（如线上传播范围、粉丝数量阈值、搜索热度等）尚无统一裁判标准。

### 1.6.3 平台经营者"优势地位"的认定标准

第15条所称"具有市场优势地位"与《反垄断法》中"市场支配地位"之间的关系，尚待司法解释厘清。学界普遍认为，"优势地位"的门槛应低于"支配地位"，但具体认定标准（市场份额阈值、依赖程度测试等）仍有争议。孔祥俊教授在《反不正当竞争法释论与适用》（2025年）中指出，第十五条须适用于特殊领域，主要指向行业准入限制、行政授权形成的优势地位，而非泛化适用。

# 第二章 各类不正当竞争行为的构成要件与司法认定

《反不正当竞争法》第二章（第7—15条）共规定了九类不正当竞争行为，构成反不正当竞争法的核心实体规范体系。2025年修订对其中多条进行了实质性修改，新增了两项全新的行为类型（平台强制低价、滥用优势地位），标志着中国反不正当竞争制度从传统商业领域向平台经济、数字经济的全面拓展。

## 2.1 混淆行为（第7条）

### 2.1.1 2025年修订要点

2025年修订对《反不正当竞争法》第七条（混淆行为）进行了重要修改，主要体现在以下三个方面：

**（1）搜索关键词条款正式入法**

2025年修订在第七条中新增搜索关键词条款，明确将"关键词"纳入混淆行为的规制对象。孔祥俊教授在《反不正当竞争法释论与适用》（2025年）中指出，该条款属于集合性和指引性条款，将各类商业标识的关键词使用行为统一归为混淆行为，结束了此前司法实践中对关键词混淆行为是否构成不正当竞争的争议。

**（2）帮助混淆行为入法**

2025年修订新增帮助混淆行为的规制条款，明确为他人实施混淆行为提供帮助的，同样构成不正当竞争。这一修改弥补了此前仅规制直接混淆行为、忽视帮助行为的制度漏洞，有助于从源头上遏制混淆行为的发生。

**（3）标识类型进一步扩展**

2025年修订将混淆行为的规制对象从原有的"商品名称、包装、装潢等"扩展至更广泛的商业标识领域，包括但不限于：商品名称、商品包装、商品装潢、企业名称（包括字号）、社会组织名称（包括简称）、姓名（包括笔名、艺名）、域名主体部分、网站名称、应用程序名称，以及2025年修订新增的搜索关键词、新媒体账号名称等。

### 2.1.2 构成要件分析

混淆行为的认定须满足以下要件：

**（1）标识的适格性**

被混淆的标识须具备"有一定影响"的显著性。根据法释〔2022〕9号第四条，"有一定影响"应同时满足两个条件：一是该标识在市场上具有一定的知名度；二是该标识具有区别商品来源的显著特征。

孔祥俊教授在《反不正当竞争法释论与适用》（2025年）中进一步指出，"有一定影响"的认定应综合考量以下因素：（1）使用该标识的商品或服务的市场销售区域；（2）该标识的宣传持续时间、范围和程度；（3）该标识是否曾被认定为知名商品或著名商标；（4）该标识所标识的商品或服务的消费群体。

**（2）混淆可能性**

混淆可能性是认定混淆行为的核心要件。法释〔2022〕9号第十二条规定，"引人误认"包括三种情形：（1）使相关公众误认为是他人商品；（2）使相关公众误认为与他人存在许可，合作，投资等特定联系；（3）使相关公众误认为他人商品与本经营者的商品存在特定联系。

孔祥俊教授强调，混淆是划分模仿自由、自由竞争与不正当竞争的根本标准。未导致混淆的模仿行为属于自由竞争范畴，反法不应过度介入。"利己不损人"的竞争行为即便在道德上值得商榷，也不构成反法意义上的不正当竞争。

**（3）混淆行为的类型**

根据2025年修订后的第七条及法释〔2022〕9号的解释，混淆行为主要包括以下类型：

**表2-1 混淆行为类型汇总**

| 行为类型 | 具体表现 | 法律依据 |
|---------|---------|---------|
| 商品名称混淆 | 擅自使用与他人商品名称近似的名称 | 第7条第1款第1项 |
| 包装装潢混淆 | 擅自使用与他人商品包装、装潢相同或近似的包装、装潢 | 第7条第1款第2项 |
| 企业名称混淆 | 擅自使用与他人企业名称（包括字号）相同或近似的名称 | 第7条第1款第3项 |
| 搜索关键词混淆 | 将他人标识设置为搜索关键词，导致搜索结果混淆 | 第7条第2款（2025年新增） |
| 新媒体账号混淆 | 擅自注册、使用与他人新媒体账号名称相同或近似的账号 | 第7条（2025年扩展） |
| 帮助混淆 | 为他人实施上述混淆行为提供帮助 | 第7条第3款（2025年新增） |

### 2.1.3 典型案例

**案例一："某牛"字号不正当竞争纠纷案**

**案号**：（2024）最高法民再224号

**裁判要点**：最高人民法院再审判决认定，登记注册行为本身属于"使用"企业名称的行为。将与他人在先注册的字号相同或近似的文字登记为企业名称，即使尚未实际使用，仍构成不正当竞争。该案确立了企业名称"登记即使用"的认定标准，有效解决了实践中"注册但不使用"规避法律责任的问题。

**案例二："引流直播带货"侵害商标权及不正当竞争纠纷案**

**案号**：（2025）浙08民终563号

**裁判要点**：浙江省温州市中级人民法院二审判决认定，短视频引流行为构成仿冒混淆。侵权人在直播带货中通过短视频将他人的注册商标作为引流工具，使消费者误以为其商品与权利人存在特定联系，构成混淆行为。同时，该案适用3倍惩罚性赔偿，判决赔偿110万元，体现了2025年修订强化知识产权保护力度的立法精神。

### 2.1.4 学术前沿：孔祥俊教授"穷尽性闭环原则"

孔祥俊教授在《反不正当竞争法释论与适用》（2025年）中系统提出了"穷尽性闭环原则"，核心观点如下：

**（1）原则内容**

属于商业标识等是否混淆的行为，均应纳入第七条衡量判断，不宜再纳入第二条一般条款的调整范围。这一原则的法理基础在于：第七条已经为混淆行为的认定提供了完整的规范框架，包括标识的适格性（"有一定影响"）、混淆可能性的判断标准、法律后果等，从体系解释的角度看，这些行为类型应当"封闭"在第七条之内，不应再通过一般条款"溢出"。

**（2）隐性使用不构成不正当竞争**

孔祥俊教授指出，搜索关键词的"隐性使用"（即仅在搜索引擎后台设置关键词，但不在搜索结果标题或描述中显示该关键词）仅触发搜索链接功能，不具有商业标识使用的法律意义，属于"利己不损人"的竞争行为，不构成不正当竞争。这一观点与法释〔2022〕9号的解释一脉相承。

**（3）商标与企业名称冲突的新变化**

2025年修订后，第七条不再区分突出使用与非突出使用，将所有企业名称混淆行为统一归入不正当竞争。孔祥俊教授认为，这一修改结束了此前"突出使用构成商标侵权、非突出使用构成不正当竞争"的二元分割格局，统一了法律适用标准。

### 2.1.5 规范衔接说明

**（1）与法释〔2022〕9号的衔接**

法释〔2022〕9号对第七条的适用提供了重要补充：第八条明确将营业形象（即商品或服务的整体营业形象，包括营业人员的服装、营业场所的装潢设计等）认定为"装潢"；第九条列举了不具有区分商品来源显著特征的标识类型；第十五条明确了帮助混淆行为的认定标准。

**（2）行政处罚参数**

**表2-2 混淆行为行政处罚参数**

| 处罚情形 | 处罚标准 | 法律依据 |
|---------|---------|---------|
| 违法经营额5倍以下罚款 | 违法经营额×5 | 第7条第1款 |
| 不足5万元处25万元以下罚款 | ≤25万元 | 第7条第1款 |
| 情节严重 | 吊销营业执照 | 第7条第2款 |

## 2.2 商业贿赂（第8条）

### 2.2.1 2025年修订要点

2025年修订对《反不正当竞争法》第八条进行了实质修改，核心变化在于建立双向禁止机制，并引入责任到人的个人责任追究制度。

**（1）双向禁止机制的确立**

2025年修订在第八条中明确规定，商业贿赂包括两种情形：一是经营者行贿（即"经营者不得采用财物或者其他手段贿赂相关单位或者个人，以谋取交易机会或者竞争优势"）；二是相关单位或个人受贿（即"相关单位或者个人不得利用职权或者影响力，索取、收受经营者的财物或者其他利益"）。

这一双向禁止机制填补了此前仅禁止行贿、疏于规制受贿的制度漏洞。孔祥俊教授在《反不正当竞争法释论与适用》（2025年）中指出，从源头上遏制商业贿赂，须同时规制行贿和受贿两端，单向规制难以取得实效。

**（2）责任到人的制度创新**

2025年修订的另一重要创新是确立了商业贿赂的个人责任：

1. **个人行贿的独立处罚**：个人实施商业贿赂行为的，可处100万元以下罚款；
2. **个人受贿的独立处罚**：个人利用职权或者影响力收受贿赂的，可处50万元以下罚款；
3. **直接责任人员的责任**：单位实施商业贿赂行为的，对直接负责的主管人员和其他直接责任人员可并处处罚。

### 2.2.2 构成要件分析

**（1）行贿行为的构成要件**

| 要件 | 内容 | 说明 |
|------|------|------|
| 主体 | 经营者 | 包括法人、非法人组织和自然人 |
| 主观要件 | 谋取交易机会或竞争优势 | 须与业务往来存在因果关系 |
| 手段 | 财物或其他利益 | 包括现金、实物、会员服务、旅游招待、股权激励等 |
| 对象 | 相关单位或个人 | "相关"指与交易活动存在关联的单位或个人 |

**（2）受贿行为的构成要件**

| 要件 | 内容 | 说明 |
|------|------|------|
| 主体 | 相关单位或个人 | 包括交易相对方的单位及其工作人员，以及能够影响交易决策的第三方 |
| 客观要件 | 利用职权或影响力索取或收受利益 | "利用职权"指利用职务便利；"影响力"指利用非职务性影响力（如亲属关系、朋友关系等） |
| 主观要件 | 故意 | 明知利益与交易存在关联而索取或收受 |

### 2.2.3 典型案例

**案例：医疗设备采购商业贿赂案**

**裁判要点**：某医疗器械公司为获得医院采购订单，向医院设备科负责人支付"学术推广费"80万元。法院认定该费用名为"学术推广"、实为商业贿赂，判决该公司及负责人承担行政处罚责任。该案体现了执法机关穿透商业贿赂伪装形式、认定实质贿赂行为的审查标准。

## 2.3 虚假宣传（第9条）

### 2.3.1 2025年修订要点

2025年修订对《反不正当竞争法》第九条（虚假宣传）进行了两处重要修改：

**（1）误导对象扩展**

2025年修订将虚假宣传的误导对象从原有的"消费者"扩展至"消费者和其他经营者"。这一修改回应了互联网经济中商业主体之间信息不对称日益突出的现实，使反法对虚假宣传的规制更加周延。

**（2）虚假评价行为入法**

2025年修订新增虚假评价（俗称"刷单炒信"）的规制条款，明确"组织虚假交易或者利用虚假或者他人的名义进行虚假评价"构成不正当竞争。这一修改结束了此前对"刷单炒信"行为是否构成虚假宣传的争议，为执法机关查处此类行为提供了明确依据。

### 2.3.2 构成要件分析

**（1）虚假宣传行为的类型**

根据第九条及法释〔2022〕9号第十六条的规定，虚假宣传包括以下类型：

1. **内容虚假**：对商品的性能、功能、质量、销售状况、用户评价、曾获荣誉等作虚假的商业宣传；
2. **引人误解**：对商品作片面的宣传或者对比；以歧义性语言或者其他引人误解的方式进行商品宣传；
3. **虚假评价**：组织虚假交易；或者利用虚假或者他人的名义进行虚假评价。

**（2）"引人误解"的四种情形**

法释〔2022〕9号第十七条列举了"引人误解"的四种典型情形：

1. 对商品作片面的宣传或者对比，足以导致相关公众对商品质量、性能、用途等产生误解；
2. 以歧义性语言或者其他引人误解的方式进行商品宣传，足以导致相关公众对商品质量、性能、用途等产生误解；
3. 使用欺骗性或者误导性的语言、文字、图片或者符号等标价，诱导他人与其交易；
4. 其他足以导致相关公众对商品质量、性能、用途等产生误解的宣传行为。

### 2.3.3 典型案例

**案例："养车服务"不正当竞争纠纷案**

**案号**：（2025）沪73民终33号

**裁判要点**：上海知识产权法院二审判决认定，被诉侵权人通过低价营销手段编造误导性信息，在平台上虚假标注低价吸引用户，实际无法提供相应服务，构成虚假宣传。同时，该行为损害了同行业竞争者的商业利益，构成商业诋毁。法院判决赔偿500万元，体现了对虚假宣传行为加大惩处力度的司法导向。

## 2.4 侵犯商业秘密（第10条）

### 2.4.1 2025年修订要点

《反不正当竞争法》第十条（侵犯商业秘密）的实体规范在2025年修订中未作重大修改，但新增的第三十九条（举证责任移转）对商业秘密侵权诉讼产生了深远影响。

**（1）第十条主要规范未变**

第十条规定了侵犯商业秘密的行为类型，包括：以盗窃、贿赂、欺诈、胁迫、电子侵入等不正当手段获取商业秘密；披露、使用或者允许他人使用上述手段获取的商业秘密；违反保密义务或者权利人要求披露、使用或者允许他人使用其掌握的商业秘密；第三人明知或者应知上述违法行为仍获取、使用或者披露商业秘密。

**（2）第三十九条举证责任移转（2025年新增）**

2025年修订新增第三十九条，在商业秘密侵权诉讼中引入举证责任移转规则。根据该条，符合法定条件的，由涉嫌侵权人证明其不存在侵权行为，否则承担不利后果。

孔祥俊教授在《反不正当竞争法释论与适用》（2025年）中指出，举证责任移转规则是商业秘密保护领域的重大制度创新，其适用条件通常包括：（1）权利人已证明其拥有合法的商业秘密；（2）权利人已证明涉嫌侵权人有渠道或机会获取该商业秘密（如曾是员工、合作伙伴等）；（3）涉嫌侵权人使用的信息与权利人的商业秘密实质相同。满足上述条件后，举证责任转移至涉嫌侵权人，由其证明其获取、使用的信息来源合法（如独立研发、反向工程、合法受让等）。

### 2.4.2 构成要件分析

**（1）商业秘密的认定**

根据《反不正当竞争法》第十条第四款及法释〔2020〕7号第一条至第六条的规定，商业秘密须同时满足以下三项要件：

| 要件 | 内容 | 说明 |
|------|------|------|
| 秘密性 | 不为公众所知悉 | 即该信息作为整体或其组成部分的组合，不是公众普遍知晓或容易获得的 |
| 价值性 | 具有商业价值 | 因属秘密而具有现实的或潜在的商业价值 |
| 保密性 | 经权利人采取保密措施 | 合法控制该信息的人根据情况采取了合理的保密措施 |

法释〔2020〕7号第一条明确列举了商业秘密的范围，包括：结构、原料、组分、配方、工艺、算法、方法、步骤、数据、计算机程序等涉及技术信息和经营信息。

**（2）侵权行为的类型**

| 侵权类型 | 行为表现 | 说明 |
|---------|---------|------|
| 非法获取 | 盗窃、贿赂、欺诈、胁迫、电子侵入等 | 以不正当手段获取权利人的商业秘密 |
| 非法披露使用 | 披露、使用或者允许他人使用 | 上述手段获取的商业秘密 |
| 违约使用 | 违反保密义务 | 违反权利人要求，披露、使用或允许他人使用 |
| 知情使用 | 明知或应知仍获取、使用或披露 | 恶意第三人责任 |

### 2.4.3 典型案例

**案例一："离心压缩机选型"软件及技术秘密侵权纠纷案**

**案号**：（2022）最高法知民终1592号

**裁判要点**：最高人民法院二审判决认定，涉案离心压缩机选型软件与基本级数据不可分割，均构成技术秘密。侵权人通过不正当手段获取并使用权利人的选型软件及基本级数据，构成侵犯商业秘密。法院判决赔偿1.66亿余元，是迄今为止商业秘密侵权案件中赔偿金额最高的判决之一。

**案例二："天然蛋白酶3"商业秘密侵权纠纷案**

**案号**：（2023）最高法知民终2913号

**裁判要点**：最高人民法院二审判决认定，虽然涉案技术方案的部分信息可以通过公开渠道获取，但将这些信息组合运用于特定技术方案的整体技术思路和方法仍具有秘密性。碎片化信息公开不否定整体技术方案的秘密性。该案对"部分公开是否导致整体丧失秘密性"这一实践难题提供了明确的裁判指引。

**案例三："变身漫画特效"不正当竞争纠纷案**

**案号**：（2023）京73民终3802号

**裁判要点**：北京知识产权法院二审判决认定，人工智能模型的参数和结构属于技术秘密范畴，受反不正当竞争法保护。侵权人通过不正当手段获取权利人AI模型的参数和结构数据，构成侵犯商业秘密。该案是全国首例明确认定AI模型参数和结构受反法保护的判决，具有重要的示范意义。

### 2.4.4 学术前沿：孔祥俊教授论惩罚性赔偿与举证责任移转

孔祥俊教授在《反不正当竞争法释论与适用》（2025年）中指出，商业秘密保护制度的完善应从以下三个维度推进：

1. **降低权利人的举证负担**：举证责任移转规则的确立是正确方向，但需注意平衡保护权利人与不妨碍正常人才流动之间的关系；
2. **强化惩罚性赔偿的适用**：故意侵权且情节严重的，应依法适用惩罚性赔偿，以增强威慑力；
3. **区分"接触+实质相似"与举证责任移转**：两者在功能上有重合，但前者是传统举证规则，后者是2025年修订新增的制度创新，应注意协调适用。

### 2.4.5 规范衔接说明

**（1）与法释〔2020〕7号的衔接**

法释〔2020〕7号是商业秘密保护领域最重要的司法解释，对商业秘密的认定标准、侵权行为的判断方法、抗辩事由等提供了详细指引。2025年修订新增的第三十九条与法释〔2020〕7号构成补充关系——第三十九条侧重举证责任分配，法释〔2020〕7号侧重实体要件认定。

**（2）与第三十九条的衔接**

第三十九条的适用需要注意：（1）仅适用于民事诉讼程序，行政执法程序不适用；（2）举证责任移转须符合法定条件，权利人仍须首先完成初步举证；（3）涉嫌侵权人可提出独立研发、反向工程、合法受让等抗辩事由。

### 2.4.7 《商业秘密保护规定》新规补充

#### （1）法规信息

《商业秘密保护规定》，国家市场监督管理总局令第126号，**2026年6月1日起施行**。共31条。

#### （2）核心创新

1. **首次将"数据""算法"纳入商业秘密保护范畴**：明确数据和算法可作为商业秘密的客体，受法律保护。这与反法第13条数据专款形成互补——数据专款从竞争法角度规制不正当获取使用数据的行为，商业秘密保护规定从商业秘密角度保护具有秘密性、保密性、价值性的数据和算法。

2. **细化远程办公/跨境协作保密措施**：明确以下措施属于"合理保密措施"：
   - **权限分级**：对不同敏感程度的数据设置不同访问权限；
   - **数据脱敏**：在共享或展示时对敏感信息进行脱敏处理；
   - **操作日志留痕**：记录所有数据访问和操作行为，可追溯。

   > **诉讼举证提示**：操作日志作为"合理保密措施"的证明材料，须同时具备三要素：①**时间戳**（精确到秒，不可篡改）；②**操作者身份**（账号实名+IP/设备标识）；③**操作内容**（涉及的具体数据字段、动作类型）。仅记录"有人访问过系统"等模糊日志难以被法院采纳为有效保密措施证据。建议采用区块链存证或可信时间戳技术强化日志证明力。

3. **明确商业秘密三大要件**：
   - **秘密性**：不为公众所知悉；
   - **保密性**：具有商业价值并经权利人采取相应保密措施；
   - **价值性**：具有现实或潜在的商业价值。

4. **四类侵权行为**：
   - 以盗窃、贿赂、欺诈、胁迫、电子侵入或者其他不正当手段获取；
   - 披露、使用或者允许他人使用以前项手段获取的商业秘密；
   - 违反保密义务或者违反权利人有关保守商业秘密的要求，披露、使用或者允许他人使用其掌握的商业秘密；
   - 第三人明知或者应知商业秘密为非法获取或违反保密义务，仍获取、使用或者披露。

5. **五种合法免责行为**：
   - **独立研发**：独立自主研究发现或开发；
   - **反向工程**：通过对公开渠道获得的产品进行拆卸、测绘、分析等获取技术信息（但产品本身系通过不正当手段获得的除外）；
   - **公知信息**：从公开渠道获取的信息；
   - **合法许可**：经权利人许可获取；
   - **其他合法途径**：法律规定的其他合法获取方式。

6. **重点领域**：聚焦生物医药、集成电路、人工智能等重点领域进行"一企一策"指导。

#### （3）数据/算法作为商业秘密的保护路径

数据和算法可通过两条路径获得保护：
- **反法第13条（数据专款）**：从竞争法角度，禁止不正当获取和使用数据；
- **反法第10条（商业秘密）+《商业秘密保护规定》**：从商业秘密角度，保护具有秘密性的数据和算法。

两者的区别：
- 数据专款保护的是"合法持有"的数据，不要求秘密性；
- 商业秘密保护要求"秘密性+保密性+价值性"三要件，门槛更高但保护力度更强（可适用惩罚性赔偿1~5倍）。

#### （4）远程办公场景下的保密措施

在远程办公、跨境协作日益普遍的背景下，《商业秘密保护规定》明确的"权限分级、数据脱敏、操作日志留痕"三项措施，为企业构建远程办公保密体系提供了明确的合规指引。企业在商业秘密侵权诉讼中主张"已采取合理保密措施"时，可援引该规定证明合规。

## 2.5 不正当有奖销售（第11条）

### 2.5.1 2025年修订要点

2025年修订对《反不正当竞争法》第十一条（不正当有奖销售）新增了信息变更限制条款：经营者不得通过变更奖品品种、数量、最高奖金额等方式，影响兑奖结果，或者对兑奖结果造成实质性影响。这一修改旨在规制实践中常见的"内定奖项""虚假抽奖"等欺骗性有奖销售行为。

### 2.5.2 构成要件分析

不正当有奖销售包括三种类型：

**（1）信息不明确型**

所设奖的种类、兑奖条件、奖金金额或者奖品等有奖销售信息不明确，影响兑奖。例如：宣称"最高奖100万元"但实际中奖概率为零；奖品数量标注"若干"导致无法核实等。

**（2）最高奖金额超标型**

最高奖金额超过5万元的抽奖式有奖销售。孔祥俊教授在《反不正当竞争法释论与适用》（2025年）中指出，该限额是强制性规定，不允许经营者通过拆分奖项、延长兑奖期限等方式规避。

**（3）信息变更型（2025年新增）**

通过变更奖品品种、数量、最高奖金额等方式，影响兑奖结果或者对兑奖结果造成实质性影响。典型表现包括：抽奖时临时更换奖品、公布中奖名单时擅自变更中奖者、限制兑奖名额等。

## 2.6 损害商业信誉（第12条）

### 2.6.1 2025年修订要点

2025年修订对《反不正当竞争法》第十二条（损害商业信誉）进行了两处重要修改：

**（1）保护对象扩展**

2025年修订将商业诋毁的保护对象从原有的"竞争对手的商业信誉、商品声誉"扩展至"其他经营者的商业信誉、商品声誉"。这一修改体现了"竞争行为本位论"的制度化——原告无需证明与被告存在竞争关系，只需证明其属于"其他经营者"即可。

**（2）指使他人禁止**

2025年修订新增"指使他人"实施商业诋毁的禁止性规定，明确即使经营者自己不实施诋毁行为，但指使、怂恿他人实施诋毁行为的，同样构成不正当竞争。这一修改堵塞了此前"通过第三方实施诋毁以规避责任"的制度漏洞。

### 2.6.2 构成要件分析

根据第十二条及法释〔2022〕9号第十九条、第二十条的规定，商业诋毁的认定须满足以下要件：

| 要件 | 内容 | 说明 |
|------|------|------|
| 行为主体 | 经营者 | 包括直接实施诋毁者和指使者 |
| 行为方式 | 编造、传播虚假信息或者误导性信息 | 包括自己实施和指使他人实施 |
| 传播对象 | 相关公众 | 信息须传播至相关公众方构成侵权 |
| 损害后果 | 损害其他经营者的商业信誉、商品声誉 | 须有损害后果或损害可能 |
| 因果关系 | 虚假信息与损害后果之间存在因果关系 | 信息传播须足以导致公众误解 |

法释〔2022〕9号第十九条还明确了传播者的共同侵权责任：经营者或者个人传播虚假信息或者误导性信息，损害竞争对手的商业信誉、商品声誉的，与实施者构成共同侵权。

## 2.7 互联网不正当竞争（第13条）

### 2.7.1 2025年修订要点

2025年修订对《反不正当竞争法》第十三条（互联网不正当竞争）进行了幅度最大的修改，主要体现在以下三个方面：

**（1）数据保护专款正式入法（最大亮点）**

2025年修订在第十三条中新增数据保护专款（第二款），明确禁止以不正当方式获取、使用其他经营者合法持有的数据。这是中国首次在反法层面确立数据的专门保护规则，孔祥俊教授在《反不正当竞争法释论与适用》（2025年）中称其为"数据保护立法的重要突破"。

**条文内容**：《反不正当竞争法》（2025修订）第十三条第二款规定："经营者不得以欺诈、胁迫、避开或者破坏技术管理措施等不正当方式，获取、使用其他经营者合法持有的数据，损害其他经营者的合法权益，扰乱市场竞争秩序。"

**构成要件**：数据保护专款的适用须同时满足以下三要素：（1）不正当手段（欺诈、胁迫、避开或破坏技术管理措施等）；（2）受保护数据的适格性（须为"合法持有的数据"）；（3）损害后果。

**豁免情形**：孔祥俊教授在《反不正当竞争法释论与适用》（2025年）中指出，数据保护专款设定了以下豁免情形，为正当的数据流通预留了制度空间：（1）可从公开渠道自由获取的数据；（2）使用数量不足以造成实质性损害；（3）不与权利人正常经营冲突；（4）以诚信方式使用的其他情形。

**（2）滥用平台规则禁止入法**

2025年修订在第十三条中新增滥用平台规则禁止条款（第三款），禁止平台经营者利用技术手段、平台规则等方式，滥用平台规则对平台内经营者在平台内的交易、交易价格等进行不合理限制或者附加不合理条件。这一条款与《电子商务法》第三十五条形成规制交叉，但侧重点不同——反法侧重于竞争秩序的维护，《电子商务法》侧重于平台内经营者权益的保护。

**（3）互联网专条整体扩充**

2025年修订对第十三条进行了整体扩充，在保留原有行为类型（强制跳转、误导卸载、恶意不兼容等）的基础上，新增数据抓取和滥用平台规则两项核心内容，使互联网专条从原则性规定向具体规则转化。

### 2.7.3 典型案例

**案例一：搬家软件"盗图抄店"不正当竞争纠纷案**

**案号**：（2024）苏民终212号

**裁判要点**：江苏省高级人民法院二审判决认定，被诉侵权人通过绕开反爬措施抓取权利人平台的用户数据和商户信息，并据此"复制"其店铺页面，造成用户实质性替代，构成不正当竞争。法院判决赔偿500万元，同时指出，反爬措施是平台经营者保护数据权益的技术手段，破坏反爬措施构成"以不正当方式获取数据"。

**案号**：（2022）粤73民终3597号

**裁判要点**：广州知识产权法院二审判决认定，第三方交易平台明知游戏打金行为非法，仍为其提供交易服务，构成帮助侵权。法院认为，明知他人行为非法而提供实质性帮助的，与直接实施侵权行为构成共同侵权，判决赔偿303万元。

### 2.7.4 学术前沿：数据抓取的正当性边界

孔祥俊教授在《反不正当竞争法释论与适用》（2025年）中系统论述了数据抓取的正当性边界：

1. **技术保护措施是数据权益的外在化**：经营者通过技术保护措施（如反爬系统、访问权限控制等）将数据权益"外在化"，使得数据的控制状态可被外界感知。破坏技术保护措施构成"以不正当方式获取数据"。

2. **公开数据不等于可自由使用**：即使数据可以通过公开渠道访问，也不意味着可以不受限制地使用和商业化利用。如果使用方式造成权利人实质性损害，仍可能构成不正当竞争。

3. **实质性替代是损害判断的关键**：判断数据抓取是否造成损害，关键在于是否造成对权利人产品或服务的实质性替代，而非仅仅增加了竞争对手的用户访问量。

### 2.7.5 第13条数据专款实务解读与典型案例

#### （1）适用四要件

北京互联网法院首例数据专款案确立了数据专款适用的四要件审查框架：

| 要件 | 内容 | 认定标准 |
|---|---|---|
| 客体要件 | 合法持有的数据 | 合法收集+实质性投入+商业价值 |
| 主体要件 | 数据持有者与竞争者 | 存在竞争关系（含潜在竞争） |
| 行为要件 | 不正当获取或使用数据 | 爬取、抓取、复制等手段 |
| 结果要件 | 损害竞争秩序或合法权益 | 实质性损害或明显损害风险 |

#### （2）"合法持有数据"的认定标准

北京互联网法院在首例数据专款案中明确了"合法持有数据"的三项认定标准：
1. **合法收集**：数据来源合法，未违反法律法规禁止性规定；
2. **实质性投入**：数据持有者对数据的收集、整理、维护投入了实质性劳动和资金；
3. **商业价值**：数据具有商业价值，能够为持有者带来竞争优势。

#### （3）典型案例——北京互联网法院首例数据专款案

- **案情**：被告王某爬取某知名职场社交平台用户数据（姓名、公司职位、工作经历、教育背景等），搭建网站售卖。
- **裁判**：法院依据反法第13条认定构成不正当竞争，判赔经济损失20万元+合理开支3万余元，共计23万余元。
- **意义**：首次适用数据专款，明确了四要件审查框架和"合法持有数据"的认定标准。

#### （4）典型案例——西城法院首例电商平台数据权益案

- **案情**：被告伪装用户获取京东平台数据。
- **裁判**：认定构成不正当竞争，判赔100万元+合理开支25.88万元，共计125.88万元。
- **意义**：首例电商平台数据权益案，体现了对平台数据资产的高强度保护。

#### （5）典型案例——代拍平台抢票案

- **案情**：代拍平台提供技术手段抢票。
- **裁判**：认定构成不正当竞争，判赔30万元+合理开支7.53万元，共计37.53万元。

## 2.8 平台强制低价（第14条）——2025年新增

### 2.8.1 条款背景

《反不正当竞争法》第十四条是2025年修订新增的条款，旨在规制平台经济中的强制低价行为。该条款的出台背景是：近年来，部分大型平台经营者利用其平台优势地位，强制平台内经营者以低于成本的价格销售商品，严重扰乱了正常的市场竞争秩序，损害了平台内经营者的合法权益。

孔祥俊教授在《反不正当竞争法释论与适用》（2025年）中指出，第十四条的制定体现了立法者对平台经济特殊性的深入认识——平台作为市场交易的组织者，其规则设计对平台内经营者的经营自由有着直接影响，平台滥用优势地位实施强制低价，本质上是一种不正当竞争行为。

### 2.8.2 构成要件分析

**表2-10 平台强制低价构成要件**

| 要件 | 内容 | 说明 |
|------|------|------|
| 主体 | 平台经营者 | 利用网络从事经营活动，为平台内经营者提供交易场所或者交易服务的经营者 |
| 手段 | 技术手段、平台规则等方式 | 包括算法推荐、搜索降权、扣减保证金、限制流量等 |
| 行为 | 强制平台内经营者以低于成本的价格销售商品 | "强制"强调行为的不合理性，"低于成本"强调行为的经济损害性 |
| 结果 | 扰乱市场竞争秩序 | 须对市场竞争秩序造成损害 |

### 2.8.3 与相关条款的区分

**（1）与《电子商务法》第三十五条的区分**

《电子商务法》第三十五条侧重于保护平台内经营者的自主经营权，规制的是平台对平台内经营者的"不合理限制"；第十四条侧重于维护市场竞争秩序，规制的是平台强制低价对市场公平竞争的破坏。两者在规制目的、适用标准和法律后果上存在差异。

**（2）与《反垄断法》的区分**

平台强制低价与《反垄断法》上的"掠夺性定价"在形式上相似，但法律适用标准不同。《反垄断法》要求证明经营者在相关市场具有支配地位；第十四条不以支配地位为要件，门槛相对较低，更有利于对平台经济中的强制低价行为及时进行规制。

### 2.8.4 行政处罚参数

**表2-11 平台强制低价行政处罚参数**

| 处罚情形 | 处罚标准 | 法律依据 |
|---------|---------|---------|
| 一般情形 | 5万元以上50万元以下罚款 | 第14条 |
| 情节严重 | 50万元以上200万元以下罚款 | 第14条 |

### 2.8.5 第14条平台强制低价实务要点

#### （1）规制行为类型

1. **直接强制**：平台通过规则明确要求经营者以低于成本价销售；
2. **变相强制**：平台通过算法降权、流量限制、搜索屏蔽等技术手段迫使经营者降价；
3. **自动调价**：平台利用自动调价助手等工具实施变相强制。

#### （2）与反垄断法的区别

- 反垄断法规制的是具有市场支配地位的经营者滥用支配地位，门槛高；
- 反法第14条规制的是平台经营者利用平台规则和技术手段实施的强制行为，不要求证明市场支配地位，门槛更低。

#### （3）实务要点

截至修订时（2026年6月20日），第14条尚无生效民事裁判文书。"变相强制"的认定标准可从"自动调价助手"入手，逐步扩展至算法降权、流量限制等场景。市监总局2026年6月16日发布的"618"合规提示中"规范补贴促销行为"直接对应反法第14条，《外卖平台补贴行为规范十条（征求意见稿）》第4条"不得导致平台内经营者以低于成本价格销售商品"亦与第14条形成衔接（详见第七章）。

### 2.8.6 信息缺口声明

第十四条为2025年新增条款，截至本报告撰写时，尚无生效裁判文书。该条款的具体适用标准（如"成本"的认定方法、"强制"的判断标准等）有待司法解释和执法实践进一步明确。

## 2.9 滥用优势地位（第15条）——2025年新增

### 2.9.1 条款背景

《反不正当竞争法》第十五条是2025年修订另一新增条款，规制具有市场优势地位的经营者滥用其优势地位实施不正当竞争的行为。该条款的出台填补了反法对"优势地位"规制的空白，与《反垄断法》形成互补。

孔祥俊教授在《反不正当竞争法释论与适用》（2025年）中指出，第十五条的制度价值在于：其一，弥补《反垄断法》适用门槛较高的不足，将部分尚未达到《反垄断法》"市场支配地位"认定标准但确实具有优势地位的行为纳入规制；其二，明确"优势地位"与"支配地位"的差异化认定标准，防止两法适用边界的模糊。

### 2.9.2 构成要件分析

**表2-12 滥用优势地位构成要件**

| 要件 | 内容 | 说明 |
|------|------|------|
| 主体 | 具有市场优势地位的经营者 | 优势地位的认定门槛低于垄断法的支配地位 |
| 行为方式 | 利用优势地位实施不正当竞争 | 须与其他条款（第七条至第十四条）规定的行为类型相结合 |
| 损害后果 | 扰乱市场竞争秩序，损害其他经营者合法权益 | 须有损害后果或损害可能 |

### 2.9.3 与《反垄断法》的区分

**表2-13 滥用优势地位与滥用市场支配地位的区分**

| 比较维度 | 第十五条（滥用优势地位） | 《反垄断法》（滥用市场支配地位） |
|---------|----------------------|----------------------------|
| 认定门槛 | 较低（"优势地位"） | 较高（"支配地位"） |
| 适用标准 | 相对灵活 | 严格（需界定相关市场、认定支配地位） |
| 举证责任 | 原告举证难度较低 | 原告举证难度较高 |
| 处罚力度 | 最高500万元（反法处罚标准） | 上一年度销售额1%~10%（反垄断法处罚标准） |
| 制度定位 | 补充《反垄断法》 | 独立规制体系 |

> **核实说明**：表2-13中"处罚力度"列比较的是两法的处罚标准差异。《反不正当竞争法》第十五条规定最高罚款500万元；《反垄断法》规定为上一年度销售额的1%至10%。两者适用不同法条，不存在矛盾。

### 2.9.4 行政处罚参数

**表2-14 滥用优势地位行政处罚参数**

| 处罚情形 | 处罚标准 | 法律依据 |
|---------|---------|---------|
| 滥用优势地位 | 限期改正；逾期不改正的，处100万元以下罚款 | 第15条 |
| 情节严重 | 100万元以上500万元以下罚款 | 第15条 |

> **核实说明**：表2-14数据与表2-13中"处罚力度"列对应关系说明——表2-13引用"最高500万元"是指情节严重时的处罚上限；表2-14分别列明一般情形（限期改正+逾期100万以下）和情节严重情形（100万~500万），两表数据一致，不存在矛盾。

### 2.9.5 第15条滥用优势地位实务要点

#### （1）核心要素

- **"大型企业"界定**：参照《中小企业划型标准规定》或出台专门界定标准，尚未有明确规则；
- **"市场优势地位"认定**：低于反垄断法"市场支配地位"门槛，但具体标准待明确；
- **行为类型**：拖欠账款为主，可能扩展至其他滥用优势地位的行为。

#### （2）实务要点

该条主要为行政执法条款，民事赔偿案例尚待发展。中小企业可援引该条向市场监督管理部门投诉，也可在民事诉讼中作为主张被告行为违法的依据。市场监管总局在2026年执法指南中已将"大型企业拖欠账款"列为滥用优势地位的重点整治领域。

### 2.9.6 信息缺口声明

第十五条为2025年新增条款，截至本报告撰写时，尚无生效裁判文书。该条款中"优势地位"的认定标准、与《反垄断法》"支配地位"的界定差异，有待司法解释和执法实践进一步明确。

## 2.10 第40条域外适用（2025年新增）

### 2.10.1 条文要旨

《反不正当竞争法》（2025修订）第四十条规定：在中华人民共和国境外实施的不正当竞争行为，扰乱境内市场竞争秩序，或者损害境内经营者、消费者合法权益的，适用本法。

### 2.10.2 适用条件

1. 行为发生在境外；
2. 扰乱境内市场竞争秩序，或损害境内经营者/消费者合法权益；
3. 适用"效果原则"（Effects Doctrine）——与美国法中的效果原则类似。

> **比较法注释**：美国《谢尔曼法》（1890年）经 *Hartford Fire Insurance Co. v. California*, 509 U.S. 764 (1993) 等判例确立"效果原则"，是普通法系域外适用的代表。中国反法第40条系首次以**制定法明文形式**确立效果原则，立法路径不同——并非单纯移植美国法，而是结合中国跨境贸易与数字竞争监管需求的独立立法选择，与《反垄断法》第2条（2008年）形成体系衔接。

### 2.10.3 与法释〔2022〕9号第27条的衔接

- 法释〔2022〕9号第27条解决的是**管辖权**问题（侵权结果发生地法院管辖）；
- 第40条解决的是**法律适用**问题（境外行为适用中国反法）；
- 两者配合，构成跨境不正当竞争纠纷的"管辖+法律适用"完整框架。

### 2.10.4 典型涉外情形

1. 境外网站实施虚假宣传（面向中国消费者）；
2. 境外爬取境内平台数据（避开技术管理措施）；
3. 境外主体侵犯中国商业秘密；
4. 境外平台强制境内商家低价（跨境电商场景）；
5. 境外商业诋毁影响境内市场（通过互联网传播）。

### 2.10.5 实务要点

截至修订时（2026年6月20日），第40条尚无域外适用的生效裁判文书。跨境案件管辖和执行依赖双边司法协助条约和互惠原则。域外适用的"初步证据"标准和"效果"认定标准有待司法实践发展。

# 第三章 典型诉讼案例深度分析

反不正当竞争诉讼是中国知识产权保护和市场竞争秩序维护的重要阵地。近年来，随着数字经济的快速发展和市场竞争的日趋激烈，反不正当竞争案件的数量持续增长，案件类型日益多样化，裁判规则也在实践中不断丰富和完善。本章以2024年至2025年最新的司法实践和统计数据为基础，系统梳理各类不正当竞争行为的典型案例，提炼裁判规则，分析学术前沿，以期为实务工作提供参考。

| 技术秘密:经营秘密案件比例 | 约2:1 | 技术秘密案件占多数 |
| 民事案件审理周期 | 平均156天 | — |

从案件类型来看，技术秘密案件与经营秘密案件的比例约为2:1，技术秘密案件占据多数，反映出企业技术创新成果保护需求的增长态势。民事案件的平均审理周期为156天，相较于其他知识产权案件较长，主要原因在于商业秘密案件的审理往往涉及复杂的技术事实认定。

孔祥俊教授在《反不正当竞争法释论与适用》（2025年）中指出，商业秘密保护的司法实践呈现以下趋势：一是举证责任移转规则的确立显著降低了权利人的诉讼门槛；二是惩罚性赔偿的适用增强了司法保护的威慑力；三是技术事实查明机制的完善提升了案件审理的质量和效率。

## 3.2 混淆行为典型案例

混淆行为是反不正当竞争法中最传统也是最常见的类型之一。2025年修订进一步扩展了混淆行为的规制范围，将搜索关键词、新媒体账号等数字标识纳入保护范畴。以下通过典型案例分析混淆行为的司法认定标准。

### 3.2.1 "某牛"字号不正当竞争纠纷案

**案号**：（2024）最高法民再224号

**案件概况**：本案是一起涉及企业名称（字号）混淆的再审案件。原告在某市登记注册了含有"某牛"文字的企业名称，被告随后在同一地区登记注册了含有相同文字的企业名称。原告主张被告的行为构成不正当竞争，请求判令被告停止使用并赔偿损失。

**裁判要点**：最高人民法院再审判决认定，登记注册行为本身属于"使用"企业名称的行为。将与他人在先注册的字号相同或近似的文字登记为企业名称，即使尚未实际使用，仍构成不正当竞争。该案确立了企业名称"登记即使用"的认定标准，有效解决了实践中"注册但不使用"规避法律责任的问题。

**裁判规则提炼**：

1. **登记即使用原则**：企业名称的登记注册行为本身即构成"使用"，无需实际开展经营活动；
2. **在先权利保护**：在先登记注册的字号具有在先权益，他人不得登记相同或近似字号；
3. **混淆可能性判断**：即使双方经营地点不同，但字号相同仍可能导致相关公众混淆。

**学术评析**：孔祥俊教授在《反不正当竞争法释论与适用》（2025年）中指出，本案确立的"登记即使用"原则具有重要的实践意义。在互联网经济时代，企业字号不仅是商业标识，更是重要的无形资产。允许他人通过登记注册方式"占有"他人字号，即使不实际使用，也会阻碍在先权利人的字号扩张和发展空间。该原则的确立，有效堵塞了通过"注册不使用"规避反法责任的制度漏洞。

### 3.2.2 "引流直播带货"侵害商标权及不正当竞争纠纷案

**案号**：（2025）浙08民终563号

**案件概况**：本案是一起涉及短视频引流行为的侵权纠纷。某电商经营者在直播带货过程中，通过短视频将他人的注册商标作为引流工具，吸引用户点击进入其店铺。权利人主张该行为构成混淆行为和商标侵权，请求适用惩罚性赔偿。

**裁判要点**：浙江省温州市中级人民法院二审判决认定，短视频引流行为构成仿冒混淆。侵权人在直播带货中通过短视频将他人的注册商标作为引流工具，使消费者误以为其商品与权利人存在特定联系，构成混淆行为。同时，该案适用3倍惩罚性赔偿，判决赔偿110万元，体现了2025年修订强化知识产权保护力度的立法精神。

**裁判规则提炼**：

1. **引流行为的商业标识使用属性**：将他人的注册商标作为引流工具使用，具有标识商品来源的功能，属于商业标识使用行为；
2. **混淆可能性的判断**：相关公众基于引流信息对商品来源产生误认，即构成混淆；
3. **惩罚性赔偿的适用**：故意实施混淆行为且情节严重的，可适用惩罚性赔偿。

**学术评析**：孔祥俊教授在《反不正当竞争法释论与适用》（2025年）中指出，本案涉及的是数字营销场景下的混淆行为认定问题。在直播电商蓬勃发展的背景下，利用他人商标进行引流的现象十分普遍。从混淆行为的构成要件来看，关键在于判断引流行为是否导致相关公众对商品来源产生误认。本案的裁判表明，即使是间接的引流行为，只要具有标识商品来源的功能，且导致相关公众混淆，仍可构成混淆行为。

综合上述案例，混淆行为的司法认定应把握以下核心规则：

| 裁判规则 | 内容 | 案例依据 |
|---------|------|---------|
| 登记即使用原则 | 企业名称登记注册行为本身即构成"使用" | "某牛"字号案 |

## 3.3 商业秘密典型案例

商业秘密是反不正当竞争法保护的重要内容。2024年至2025年，最高人民法院审理了多起具有示范意义的商业秘密案件，其中新能源汽车底盘技术秘密案以6.4亿余元的赔偿金额创下了商业秘密案件赔偿的新纪录。

### 3.3.1 新能源汽车底盘技术秘密案——6.4亿余元惩罚性赔偿

**案件概况**：本案是一起涉及新能源汽车底盘技术秘密的侵权纠纷。涉案技术秘密包括新能源汽车底盘的核心技术方案、工艺参数等。侵权人通过有组织、有计划地挖取技术人才，获取并使用了权利人的技术秘密。

**裁判日期**：2024年6月14日（二审判决）

**裁判法院**：最高人民法院知识产权法庭

**核心数据**：
- 涉案技术人员约40人
- 涉案实用新型专利12件
- 二审赔偿约6.4亿余元
- 利润贡献率8%
- 惩罚性赔偿倍数2倍

**裁判要点**：

1. **举证责任整体分析法**

最高人民法院在本案中创立了举证责任整体分析法：当被诉侵权人违背研发规律、明显短于独立研发所需时间、有渠道获取技术秘密时，可以推定侵权成立。这一方法有效缓解了商业秘密权利人在举证方面的困难。

2. **惩罚性赔偿适用条件**

二审法院认定，侵权人有组织、有计划以不正当手段大规模挖取技术人才及资源的行为，构成"故意+情节严重"，依法适用惩罚性赔偿。最终确定的惩罚性赔偿倍数为2倍。

3. **停止侵害责任细化**

针对技术秘密案件的特殊性，法院对停止侵害责任进行了细化，具体包括：
- 技术秘密的管理和载体销毁
- 专利处分限制
- 客户通知
- 人员警示

4. **迟延履行金制度**

法院创设了迟延履行金制度：
- 违反停止使用义务每日100万元
- 擅自处分专利每件100万元

**学术评析**：孔祥俊教授在《反不正当竞争法释论与适用》（2025年）中高度评价本案的意义，指出：

1. **举证责任整体分析法**是商业秘密司法保护的重要创新。传统举证规则要求权利人逐一证明侵权人的侵权手段，但实践中侵权人往往通过口头传授、经验交流等方式实施侵权，举证极为困难。整体分析法的引入，使得法院可以综合考量研发时间、渠道便利性、技术相似度等因素进行整体判断，有效缓解了举证困难。

2. **迟延履行金制度**是对传统停止侵害责任的重要发展。在技术秘密案件中，侵权人往往通过继续使用侵权技术来维持竞争优势，单纯判令停止使用难以发挥实际效果。迟延履行金制度的引入，使得继续侵权的成本大幅提高，有效威慑了持续侵权行为。

3. **惩罚性赔偿的适用**体现了对恶意侵权行为的严厉打击。本案中，有组织、有计划挖取技术人才的行为具有明显的恶意，依法适用惩罚性赔偿，符合2025年修订强化商业秘密保护的立法精神。

### 3.3.2 "离心压缩机选型"软件及技术秘密侵权纠纷案

**案号**：（2022）最高法知民终1592号

**案件概况**：本案涉及离心压缩机选型软件和技术秘密的保护问题。原告主张其开发的离心压缩机选型软件及基本级数据构成技术秘密，被告通过不正当手段获取并使用了该软件和数据。

**裁判要点**：最高人民法院二审判决认定，涉案离心压缩机选型软件与基本级数据不可分割，均构成技术秘密。侵权人通过不正当手段获取并使用权利人的选型软件及基本级数据，构成侵犯商业秘密。法院判决赔偿1.66亿余元，是迄今为止商业秘密侵权案件中赔偿金额最高的判决之一。

**裁判规则提炼**：

1. **软件与数据的不可分割性**：选型软件与基本级数据在功能上相互依存，不可分割，整体构成技术秘密；
2. **数据的技术秘密属性**：用于支撑软件运行的基础数据，在与软件结合使用时具有独立的技术价值，可构成技术秘密；
3. **损害赔偿的计算**：以侵权人侵权获利为依据，结合技术秘密对产品利润的贡献率确定赔偿数额。

### 3.3.3 "天然蛋白酶3"商业秘密侵权纠纷案

**案号**：（2023）最高法知民终2913号

**案件概况**：本案涉及涉案技术方案秘密性的判断问题。被告主张，涉案技术方案的部分信息可以通过公开渠道获取，因此整体不具有秘密性。

**裁判要点**：最高人民法院二审判决认定，虽然涉案技术方案的部分信息可以通过公开渠道获取，但将这些信息组合运用于特定技术方案的整体技术思路和方法仍具有秘密性。碎片化信息公开不否定整体技术方案的秘密性。

**裁判规则提炼**：

1. **整体性判断原则**：商业秘密的秘密性应从整体技术方案的角度进行判断，而非逐一分析各组成部分；
2. **组合的非显而易见性**：将公开信息进行特定组合，形成具有非显而易见性的技术方案，该组合整体可构成商业秘密；
3. **部分公开与整体秘密的关系**：部分信息可从公开渠道获取，不必然导致整体技术方案丧失秘密性。

**学术评析**：孔祥俊教授在《反不正当竞争法释论与适用》（2025年）中指出，本案对"部分公开是否导致整体丧失秘密性"这一实践难题提供了明确的裁判指引。在技术研发过程中，完全从零开始的创新极为罕见，大多数技术创新都是建立在现有技术的基础之上。本案的裁判表明，只要整体技术方案具有非显而易见性，即使部分信息来源于公开渠道，仍可获得商业秘密保护。

### 3.3.4 商业秘密裁判规则总结

**表3-5 商业秘密裁判规则汇总**

| 裁判规则 | 内容 | 案例依据 |
|---------|------|---------|
| 举证责任整体分析法 | 综合考量研发时间、渠道便利性、技术相似度等因素推定侵权 | 新能源汽车底盘案 |
| 迟延履行金制度 | 违反停止使用义务每日100万元；擅自处分专利每件100万元 | 新能源汽车底盘案 |

### 3.3.5 吉利诉威马案（6.4亿元标杆案）

#### （1）案件信息

- **案号**：（2023）最高法知民终1590号
- **审理法院**：最高人民法院知识产权法庭（二审/终审）
- **判决作出日期**：2024年4月25日
- **判决发布日期**：2024年6月14日
- **当事人**：上诉人（一审原告）浙江吉利控股集团、吉利汽车研究院；被上诉人（一审被告）威马汽车科技集团及关联公司
- **案件性质**：侵害技术秘密纠纷（新能源汽车底盘技术）
- **索赔金额**：21亿元（中国知识产权诉讼史上最高索赔额）

#### （2）案件事实

2016—2018年间，吉利控股集团下属成都高某汽车近40名高级管理人员及技术人员（其中30人于2016年集中离职）相继离职赴威马集团及其关联公司工作。离职人员带走吉利在新能源汽车底盘、电池、电机等领域的核心技术资料。2018年吉利提起诉讼，主张威马EX5车型部分技术来源于其商业秘密。

#### （3）裁判结果

最高人民法院二审改判，判决威马方赔偿吉利合计 **约6.4亿余元**，创我国知识产权侵权诉讼判赔数额历史最高纪录。具体构成：

| 项目 | 金额 | 计算方法 |
|------|------|---------|
| 侵权获利（基数） | 约5亿元 | 威马EX5整车利润 × 技术贡献率8% |
| 惩罚性赔偿 | 2倍 | 基数×2 |
| 维权合理开支 | 约1.4亿元 | 律师费+调查费等 |
| **合计** | **约6.4亿元** | — |

#### （4）技术贡献率8%完整计算链

本案以"威马EX5整车利润×技术贡献率8%"为侵权获利计算基数，技术贡献率8%的计算链如下：

| 计算层级 | 参数 | 数值 | 说明 |
|---|---|---|---|
| 第一层 | 整车技术利润分成率 | 25% | 参考联合国工业发展组织标准16%~27%，取25% |
| 第二层 | 底盘技术占整车技术权重 | 40% | 发动机45%/底盘40%/车身5%/电气10% |
| 第三层 | 底盘技术对整车销售利润贡献率 | 10% | 25% × 40% = 10% |
| 第四层 | 技术秘密占底盘技术比例 | 80% | 吉利方主张，法院采纳 |
| **最终** | **技术秘密对整车销售利润贡献率** | **8%** | **10% × 80% = 8%** |

该方法解决了"整车利润中属于商业秘密的部分"难以精确计算的难题，为复合型产品中提取商业秘密价值占比提供了可援引的计算范式。

#### （5）惩罚性赔偿分段适用

本案惩罚性赔偿的适用采用分段计算方法：

| 时间段 | 适用规则 | 说明 |
|---|---|---|
| 2018.9—2019.4 | 仅补偿性赔偿 | 2019年4月修正反法尚未施行，无惩罚性赔偿依据 |
| 2019.5—2022.Q1 | 适用2倍惩罚性赔偿 | 2019年修正反法施行后，符合惩罚性赔偿条件 |

吉利方原主张5倍惩罚性赔偿，法院综合考量威马侵权主观过错程度（有组织、有计划挖角）及情节严重程度（大规模、持续侵害），酌定2倍，体现"过罚相当"原则。该分段适用方法与法释〔2026〕7号第11条"倍数综合考量+可非整数"规则相印证。

#### （6）核心裁判规则

1. **"故意+情节严重"认定**：最高法认定威马行为构成"有组织、有计划、大规模侵害技术秘密"，符合惩罚性赔偿适用条件。该项认定与法释〔2026〕7号第7条第4项"以侵权为业"的认定标准高度契合。

2. **"停止侵害"责任的创新细化**：
   - 判决威马方不仅须立即停止使用涉密技术；
   - **创新判项**：如未在规定期限内停止侵害，每逾期一日支付 **100万元迟延履行金**；
   - 这是国内首次在知识产权案件中明确"停止侵害"义务的迟延履行金计算标准，破解了"停止侵害判决执行难"的实务困境。

#### （7）与法释〔2026〕7号的对照意义

本案于2024年6月判决，发生在法释〔2026〕7号（2026年5月1日施行）之前，但其裁判规则与法释〔2026〕7号高度契合，是法释〔2026〕7号第5—13条规则在商业秘密领域的标杆实践：

| 法释〔2026〕7号条文 | 本案对应体现 |
|---|---|
| 第5条（反法领域仅商业秘密侵权适用惩罚性赔偿） | 本案系技术秘密侵权，适用惩罚性赔偿，符合第5条 |
| 第6条第3、5项（"故意"认定：基于合作关系接触+实施盗版假冒类） | 威马通过挖角吉利技术人员获取技术秘密，符合第6条第3项 |
| 第7条第4项（"以侵权为业"） | 法院认定"有组织、有计划、大规模侵害" |
| 第8条（计算基数：实际损失/违法所得/侵权获利） | 本案以侵权获利为基数 |
| 第9条（以侵权为业的参照销售利润） | 本案以整车销售利润计算 |
| 第11条（倍数综合考量+可非整数） | 适用2倍整数，综合考量主观过错与情节严重 |
| 第12条（最高5倍+合理开支另行计算） | 2倍未超上限，合理开支1.4亿元另行计算 |

#### （8）对反法诉讼的实务启示

1. **商业秘密案件的天花板已被打破**：6.4亿元判赔彻底改变"商业秘密案件判赔低"的传统认知。法释〔2026〕7号施行后，类似案件赔偿上限可至基数×5倍+合理开支。

2. **"技术贡献率"方法的可援引性**：在复合型产品（整车/平台/系统）中提取商业秘密价值占比的"技术贡献率"方法，可推广至数据专款（第13条）、平台不正当竞争（第14条）等新条文案件的赔偿计算。

3. **"停止侵害+迟延履行金"组合策略**：原告在反法诉讼中可主动请求法院判令迟延履行金条款（参照本案100万元/日标准），强化停止侵害判决的执行力。

4. **挖角型商业秘密侵权的举证重点**：
   - 离职人员名册、入职竞品公司时间；
   - 离职前后的技术资料访问、复制记录（参见2.4.7节"操作日志三要素"）；
   - 竞品产品与原公司技术的实质性相似比对。

### 3.3.6 潜意识剽窃型商业秘密侵权——莱顿宣言启示

> **理论补充背景**：2026年6月2日，国际数学联盟（IMU）正式背书发布《人工智能与数学莱顿宣言》（Leiden Declaration on AI and Mathematics），由16位来自华盛顿大学、哥伦比亚大学、剑桥大学、牛津大学、莱顿大学等全球顶尖高校的数学专家起草，截至本报告定稿时已有2411位数学家签署。宣言六项核心警示中，第二项明确指出"AI模型依托海量已发表数学文献训练，多数训练未遵循原论文版权与引用协议，生成内容整合前人成果却无法精准标注出处，动摇传统署名引证规则"。该警示所揭示的"无意识整合他人智力成果"现象，在商业秘密法理上对应**潜意识剽窃型侵权**这一长期被忽视的侵权类型。

#### （1）潜意识剽窃的法理界定

**概念定义**：潜意识剽窃（Subconscious Plagiarism / Cryptomnesia），又称"潜忆抄袭""隐性记忆复制"，指行为人在不知情状态下，将既往接触过的他人智力成果（包括商业秘密信息）作为自己独立创作的成果进行使用、披露或授权他人使用的现象。

**与一般商业秘密侵权的区分**：

| 维度 | 一般商业秘密侵权 | 潜意识剽窃型侵权 |
|------|----------------|-----------------|
| 主观状态 | 明知或应知系他人商业秘密 | **无主观明知**，行为人确信系自己独立创作 |
| 接触方式 | 主动获取（挖角、爬取、盗窃等） | **被动接触**（会议交流、文档传阅、AI训练输出） |
| 行为表现 | 主动使用、披露 | 自主研发路径输出"独立成果" |
| 传统认定 | 易识别、易举证 | 难识别、举证困难 |
| 责任基础 | 主观过错责任 | **严格责任的延伸适用** |

#### （2）莱顿宣言揭示的新型风险场景

宣言指出的"AI模型依托海量文献训练→生成内容整合前人成果→无法精准标注出处"链条，对应到反法领域形成三类新型侵权场景：

**场景一：AI辅助研发中的潜意识剽窃**

| 阶段 | 行为 | 商业秘密侵权风险 |
|------|------|----------------|
| AI训练 | 模型训练数据中混入他人商业秘密信息 | 训练方不知情，但已构成"以其他不正当手段获取" |
| AI推理 | 模型生成内容与训练集中的商业秘密信息实质性相似 | 使用方不知情，但已构成"使用" |
| AI输出应用 | 输出成果被作为"自主研发成果"投入产品 | 误归因的"独立研发"抗辩不能成立 |

**场景二：跨企业人才流动中的潜忆复制**

吉利诉威马案（3.3.5节）的近40名技术人员离职，从法理上看：若该批人员在威马研发过程中**主观上并未主动复制吉利图纸**，但凭记忆"独立画出"与吉利底盘高度相似的设计图——这一情形在传统反法框架下因无主观故意而难以认定侵权，但实际损害后果与故意侵权无异。莱顿宣言的署名体系警示为这类"无主观明知但有实质损害"的案件提供了理论支撑。

**场景三：学术—产业合作中的无意识传递**

数学家与企业合作研究，研究过程中接触企业未公开技术信息，回到学术界后在论文、报告中以"自主研究成果"形式披露该等信息。该场景下数学家无主观披露恶意，但已构成商业秘密泄露。

#### （3）潜意识剽窃型商业秘密侵权的构成要件

针对上述新型场景，本报告提出五要件认定框架：

| 要件 | 内容 | 举证要点 |
|------|------|---------|
| **接触要件** | 行为人确实接触过原告商业秘密信息（含经AI训练集传递的间接接触） | 会议记录、邮件往来、AI训练日志、文档访问记录 |
| **实质性相似** | 被控侵权成果与原告商业秘密信息构成实质性相似 | 技术鉴定、特征点比对、代码相似度分析 |
| **独立研发抗辩排除** | 行为人不能证明系独立研发 | 研发时间线、原始工作记录、第三方证人 |
| **无主观明知** | 行为人确信系自己创作（这是与一般侵权的区分点，也是争议点） | 行为人陈述、合作方证言、研发日志 |
| **损害后果** | 原告商业秘密价值受损或竞争秩序被扰乱 | 市场份额变化、商业秘密价值评估 |

#### （4）责任承担的特殊性

潜意识剽窃型侵权的责任承担应在传统反法框架下作适当调适：

1. **责任基础**：原则上适用反法第10条第3款"第三人明知或者应知"中的"应知"标准——即客观上应当知道所使用信息可能来源于他人商业秘密。AI输出场景下，使用方对AI生成内容的来源负有合理审查义务。

2. **赔偿计算**：
   - 不适用惩罚性赔偿（不符合法释〔2026〕7号第6条"故意"认定标准）；
   - 适用填平原则，赔偿数额=原告实际损失或被告侵权获利；
   - 可酌情降低赔偿数额（一般可在正常侵权赔偿数额的30%~50%范围内酌减），体现主观过错程度的差异。

3. **停止侵害**：
   - 必须判令停止使用、披露；
   - 已投入产品的，应给予合理过渡期（参照吉利诉威马案的"迟延履行金"规则反向适用——给予被告合理整改期，逾期才适用迟延履行金）。

4. **署名/归属修正**：
   - 法院可判令被告公开发表声明，更正成果归属；
   - 已发表的论文、专利申请应予更正或撤回。

#### （5）对企业合规的启示

1. **AI使用前的训练数据审查义务**：企业使用第三方AI模型进行研发前，应审查模型训练数据来源的合法性，留存审查记录。未尽审查义务的，可能被认定为"应知"而承担侵权责任。

2. **跨企业人才入职的"清洁室"制度**：核心技术人员入职竞品公司后，应设立6~12个月的"清洁室期"，期间研发产出须经独立技术审查（参照3.3.5节吉利诉威马案"实质性相似比对"方法），证明系独立创作而非潜忆复制。

3. **学术—产业合作的隔离机制**：参与企业合作研究的学者，应在合作结束后签署"无意识披露防范承诺书"，承诺在合作结束后12个月内发表的论文须经企业技术审查，防止无意识披露企业商业秘密。

4. **AI生成内容的归属标注义务**：参照莱顿宣言第三章第1项"如实标注所用工具"的准则，企业使用AI生成的技术成果应标注AI工具及训练来源，便于发生争议时溯源。

#### （6）对反法第10条第3款"应知"的扩张解释建议

现行反法第10条第3款将"应知"作为第三人侵权的主观要件。在AI时代背景下，建议司法实践对"应知"作如下扩张解释：

- **传统场景**：第三人仅在明知商业秘密来源的情况下使用，才构成侵权；
- **AI时代扩张**：第三人使用AI生成内容投入商业活动时，对生成内容的来源负有合理审查义务。未尽审查义务且事后证明生成内容来源于他人商业秘密的，应认定为"应知"，构成侵权。

该项扩张解释与莱顿宣言第二章第2项"AI生成内容整合前人成果却无法精准标注出处，动摇传统署名引证规则"的警示相呼应，是宣言精神在反法实务中的转化路径。

## 3.4 商业诋毁典型案例

商业诋毁是损害竞争对手商业信誉和商品声誉的不正当竞争行为。2025年修订将商业诋毁的保护对象从"竞争对手"扩展至"其他经营者"，并新增了指使他人诋毁的禁止性规定，体现了"竞争行为本位论"的制度化。

### 3.4.1 "养车服务"不正当竞争纠纷案

**案号**：（2025）沪73民终33号

**案件概况**：本案是一起涉及虚假宣传和商业诋毁的不正当竞争纠纷。原告和被告均为汽车后市场服务提供商，被告通过低价营销手段编造误导性信息，并在平台上虚假标注低价吸引用户，实际无法提供相应服务。同时，被告向相关公众散布贬损原告服务质量的虚假信息。

**裁判要点**：上海知识产权法院二审判决认定：

1. **虚假宣传成立**：被告通过低价营销手段编造误导性信息，在平台上虚假标注低价吸引用户，实际无法提供相应服务，构成虚假宣传。

2. **商业诋毁成立**：被告向相关公众散布贬损原告服务质量的虚假信息，构成商业诋毁。

3. **竞合处理**：虚假宣传与商业诋毁在特定情形下可以并存，经营者通过编造误导性信息损害竞争对手商誉的，应同时承担虚假宣传和商业诋毁的法律责任。

4. **赔偿金额**：法院判决赔偿500万元，体现了对虚假宣传和商业诋毁行为加大惩处力度的司法导向。

**裁判规则提炼**：

1. **虚假宣传与商业诋毁的竞合**：通过虚假信息损害竞争对手商誉的，可能同时构成虚假宣传和商业诋毁；
2. **误导性信息的商业诋毁属性**：即使信息未直接指向特定竞争对手，但足以使公众对竞争对手的服务质量产生负面评价的，构成商业诋毁；
3. **损害赔偿的计算**：综合考量侵权行为的性质、持续时间、影响范围等因素确定赔偿数额。

**学术评析**：孔祥俊教授在《反不正当竞争法释论与适用》（2025年）中指出，本案涉及的是互联网平台上商业诋毁行为的新形态。在传统商业模式下，商业诋毁通常表现为直接贬损竞争对手的商品或服务；在互联网平台上，商业诋毁往往通过虚假宣传的方式间接实施。本案的裁判表明，虚假宣传与商业诋毁并非互斥关系，当同一行为同时符合两者的构成要件时，应分别评价、合并处理。

### 3.4.2 商业诋毁裁判规则总结

**表3-6 商业诋毁裁判规则汇总**

| 裁判规则 | 内容 | 案例依据 |
|---------|------|---------|
| 虚假宣传与商业诋毁竞合 | 虚假信息损害竞争对手商誉可能同时构成两罪 | 养车服务案 |
| 误导性信息的诋毁属性 | 间接贬损信息足以使公众产生负面评价即构成诋毁 | 养车服务案 |
| 损害赔偿综合考量 | 综合侵权行为性质、持续时间、影响范围等因素确定赔偿 | 养车服务案 |

## 3.5 网络不正当竞争典型案例

网络不正当竞争是数字经济时代最具代表性的不正当竞争形态。2025年修订对第十三条进行了大幅扩充，新增数据保护专款和滥用平台规则禁止条款，标志着网络不正当竞争规制体系的进一步完善。

### 3.5.1 搬家软件"盗图抄店"不正当竞争纠纷案

**案号**：（2024）苏民终212号

**案件概况**：本案是一起涉及数据抓取的不正当竞争纠纷。原告是一家搬家服务平台的运营者，被告是一家同类平台的运营者。被告通过绕开反爬措施抓取了原告平台的用户数据和商户信息，并据此"复制"了原告的店铺页面。

**裁判要点**：江苏省高级人民法院二审判决认定：

1. **反爬措施的法律意义**：反爬措施是平台经营者保护数据权益的技术手段，破坏反爬措施构成"以不正当方式获取数据"。

2. **实质性替代的认定**：被告抓取数据后用于"复制"原告店铺页面，造成用户实质性替代，构成不正当竞争。

3. **损害赔偿**：法院判决赔偿500万元。

**裁判规则提炼**：

1. **反爬措施与数据权益保护**：经营者通过技术保护措施将数据权益"外在化"，破坏技术保护措施构成不正当手段；

2. **实质性替代的判断标准**：判断数据抓取是否造成损害，关键在于是否造成对权利人产品或服务的实质性替代；

3. **数据抓取的正当性边界**：即使数据可以通过公开渠道访问，也不意味着可以不受限制地商业化利用。

**学术评析**：孔祥俊教授在《反不正当竞争法释论与适用》（2025年）中系统论述了数据抓取的正当性边界：

1. **技术保护措施是数据权益的外在化**：经营者通过反爬系统、访问权限控制等技术措施将数据权益"外在化"，使得数据的控制状态可被外界感知。破坏技术保护措施获取数据，构成"以不正当方式获取数据"。

2. **公开数据不等于可自由使用**：即使数据可以通过公开渠道访问，也不意味着可以不受限制地商业化利用。如果使用方式造成权利人实质性损害，仍可能构成不正当竞争。

3. **实质性替代是损害判断的关键**：判断数据抓取是否造成损害，关键在于是否造成对权利人产品或服务的实质性替代，而非仅仅增加了竞争对手的用户访问量。

### 3.5.2 网络游戏第三方交易平台不正当竞争纠纷案

**案号**：（2022）粤73民终3597号

**案件概况**：本案是一起涉及帮助侵权的不正当竞争纠纷。原告是一家网络游戏运营者，被告是一家第三方游戏交易平台。被告明知游戏打金行为非法，仍为其提供交易服务。

**裁判要点**：广州知识产权法院二审判决认定，第三方交易平台明知游戏打金行为非法，仍为其提供交易服务，构成帮助侵权。法院认为，明知他人行为非法而提供实质性帮助的，与直接实施侵权行为构成共同侵权，判决赔偿303万元。

**裁判规则提炼**：

1. **帮助侵权的认定**：明知他人行为非法，仍提供实质性帮助的，与直接实施侵权行为构成共同侵权；

2. **平台注意义务**：平台经营者对其用户实施侵权行为负有注意义务，明知或应知用户侵权而不采取措施的，构成帮助侵权；

3. **实质性帮助的判断**：是否提供实质性帮助，应综合考量帮助行为与侵权结果之间的因果关系。

### 3.5.3 "变身漫画特效"不正当竞争纠纷案

**案号**：（2023）京73民终3802号

**案件概况**：本案是一起涉及人工智能模型保护的不正当竞争纠纷。原告是一家AI图像生成服务的运营者，被告通过不正当手段获取了原告AI模型的参数和结构数据。

**裁判要点**：北京知识产权法院二审判决认定：

1. **AI模型参数和结构的秘密属性**：人工智能模型的参数和结构属于技术秘密范畴，受反不正当竞争法保护。

2. **不正当获取的认定**：被告通过不正当手段获取权利人AI模型的参数和结构数据，构成侵犯商业秘密。

3. **第二条一般条款的适用**：在AI模型参数和结构尚未被明确纳入商业秘密法定范围的情况下，可通过第二条一般条款给予补充保护。

**裁判规则提炼**：

1. **新技术成果的商业秘密保护**：新技术成果（如AI模型）只要符合商业秘密的秘密性、价值性、保密性三要件，即可获得商业秘密保护；

2. **参数与结构的可分离性**：AI模型的参数和结构在技术上可分离分析，可分别认定其是否具有秘密性；

3. **一般条款的补充适用**：在新技术成果尚未被明确纳入专门法保护范围时，可通过反法一般条款给予补充保护。

**学术评析**：孔祥俊教授在《反不正当竞争法释论与适用》（2025年）中指出，本案是全国首例明确认定AI模型参数和结构受反法保护的判决，具有重要的示范意义。在人工智能时代，AI模型及其训练数据已成为企业重要的竞争资源。本案的裁判表明，反不正当竞争法作为"孵化器"功能，可以为新技术成果提供过渡性保护，待条件成熟后再纳入专门法的保护范围。

## 3.7 本章小结

本章系统梳理了反不正当竞争诉讼的典型案例和裁判规则，主要结论如下：

**一、诉讼总体情况**

1. 2024年度全国查处不正当竞争案件1.42万件，罚没款总额8.05亿元，其中网络不正当竞争案件占36.4%；
2. 惩罚性赔偿制度适用案件同比增长44.2%，体现了司法保护力度的增强；
3. 商业秘密案件原告胜诉率从9.27%提升至27.4%，与举证责任移转规则和惩罚性赔偿制度的引入密切相关。

**二、混淆行为**

1. 企业名称"登记即使用"原则确立，有效堵塞了注册但不使用的规避漏洞；
2. 引流行为具有商业标识使用属性，可构成混淆行为；
3. 惩罚性赔偿适用于故意混淆且情节严重的情形。

**三、商业秘密**

1. 新能源汽车底盘案确立了举证责任整体分析法和迟延履行金制度，具有里程碑意义；
2. 碎片化信息公开不否定整体技术方案的秘密性；
3. 软件与数据的不可分割性决定了两者可整体构成技术秘密。

**四、商业诋毁**

1. 虚假宣传与商业诋毁在特定情形下可以竞合；
2. 误导性信息即使未直接指向特定竞争对手，仍可构成商业诋毁。

**五、网络不正当竞争**

1. 反爬措施是数据权益保护的技术手段，破坏反爬措施构成不正当手段；
2. 实质性替代是数据抓取损害赔偿判断的关键标准；
3. AI模型参数和结构可构成技术秘密，受反法保护。

# 第四章 诉讼策略与举证要点

反不正当竞争诉讼涉及实体认定与程序推进的双重挑战。从诉讼策略的角度，原告和被告各自面临不同的举证压力和抗辩空间。本章从原告起诉策略、举证责任分配、被告抗辩策略、行为保全四个维度，系统梳理反不正当竞争诉讼的实务要点，并结合2025年修订的新变化，分析新法对诉讼策略的影响。

## 4.1 诉讼策略概述

### 4.1.1 原告起诉策略的基本框架

反不正当竞争诉讼的原告起诉策略，应围绕以下框架展开：

**（1）诉因选择**

诉因选择是诉讼策略的起点。原告应根据案件事实，选择最有利于自身权益保护的诉因。常见的诉因包括：

1. **具体条款诉因**：选择《反不正当竞争法》第七条至第十五条中与案件事实最相匹配的具体条款作为请求权基础；
2. **一般条款诉因**：在无具体条款可适用时，可考虑以第二条一般条款作为诉因，但须满足法释〔2022〕9号第一条规定的补充性适用条件；
3. **竞合诉因**：同一行为可能同时违反多个条款，原告可选择竞合诉因以扩大权利保护范围。

**（2）管辖选择**

根据《民事诉讼法》及法释〔2022〕9号的规定，反不正当竞争案件由侵权行为地或者被告住所地人民法院管辖。原告在选择管辖法院时，应综合考量以下因素：

1. **法院的专业能力**：知识产权法院或设有知识产权审判庭的法院具有更强的专业能力；
2. **地域因素**：部分法院对特定类型案件具有丰富的审判经验；
3. **诉讼成本**：管辖法院与证据材料、当事人的距离影响诉讼成本。

**级别管辖——修订后的管辖体系**：

| 审级 | 管辖法院 | 案件范围 |
|---|---|---|
| 互联网法院 | 北京、杭州、广州互联网法院 | 三市辖区内基层法院管辖的**网络不正当竞争纠纷**（法释〔2025〕14号，2025年11月1日起） |
| 基层法院 | 全国556家具有知识产权案件管辖权的基层法院 | 诉讼标的额较低的不正当竞争案件（非网络类型或非三市辖区） |
| 中级法院 | 各地中级人民法院 | 诉讼标的额较大、跨区域的案件 |
| 知识产权法院 | 北京、上海、广州、海南知识产权法院 | 技术类商业秘密案件、重大网络不正当竞争案件（二审或重大一审） |
| 高级法院 | 各省/自治区/直辖市高级人民法院 | 重大、复杂的二审案件 |

**地域管辖——互联网法院集中管辖规则**：

法释〔2022〕9号第26条规定，因不正当竞争行为提起的民事诉讼，由侵权行为地或者被告住所地人民法院管辖。网络收货地排除规则：仅以网络购买者可以任意选择的收货地作为侵权行为地的，人民法院不予支持。

**新增——互联网法院集中管辖规则（法释〔2025〕14号）**：北京市、杭州市、广州市辖区内应由基层法院审理的网络不正当竞争纠纷，分别由三家互联网法院集中管辖。当事人可依法协议约定与争议有实际联系地点的互联网法院管辖，但格式条款须符合提示说明义务要求。

**新增——专门管辖优先规则（法释〔2025〕15号）**：协议管辖不得违反专门管辖规定。当事人不能通过管辖协议将互联网法院专属管辖的网络不正当竞争案件约定由非互联网法院审理。

**管辖选择四大策略要素**：

| 策略要素 | 说明 | 实务考量 |
|---|---|---|
| 被告所在地 | 被告住所地法院管辖 | 便于送达和执行，但可能面临地方保护 |
| 互联网法院优先 | 三市辖区网络不正当竞争案件 | 互联网法院专业化程度高，但案件量大 |
| 知识产权法院/法庭 | 北京、上海、广州知产法院审理水平领先 | 非网络类型案件或标的额较大时可选 |
| 保全难易 | 不同法院对行为保全审查标准不同 | 商业秘密案件宜选择行为保全经验丰富的法院 |
| 处理时效 | 不同法院案件积压程度不同 | 案件量大的法院审理周期可能较长 |

**主要平台管辖法院参考**：

| 平台 | 平台经营者住所地 | 可选管辖法院（2025年11月后） |
|---|---|---|
| 淘宝/天猫 | 浙江省杭州市 | 杭州中院、**杭州互联网法院**（网络不正当竞争） |
| 1688 | 浙江省杭州市 | 杭州中院、**杭州互联网法院**（网络不正当竞争） |
| 拼多多 | 上海市 | 上海知产法院、上海相关基层法院 |
| 京东 | 北京市 | 北京知产法院、**北京互联网法院**（网络不正当竞争） |
| 抖音 | 北京市 | 北京知产法院、**北京互联网法院**（网络不正当竞争） |

> **注意**：平台经营者住所地可能变更，以实际查询为准。网络不正当竞争纠纷在三市辖区的基层法院管辖案件，自2025年11月1日起统一由互联网法院管辖。

**域外管辖补充**：不正当竞争行为发生在中华人民共和国境外，但侵权结果发生在境内的，由侵权结果发生地人民法院管辖（法释〔2022〕9号第27条）。该条与2025年反法新增第40条（域外适用效力）相互配合，共同构建了跨境不正当竞争纠纷的管辖框架（详见2.10节）。

**（3）诉讼请求设计**

诉讼请求的设计应涵盖以下层次：

1. **停止侵害请求**：请求判令被告停止正在实施的不正当竞争行为；
2. **损害赔偿请求**：请求判令被告赔偿经济损失及合理开支；
3. **消除影响请求**：请求判令被告在一定范围内消除因其不正当竞争行为造成的不良影响；
4. **销毁侵权物品请求**：在特定案件中，请求销毁用于实施不正当竞争的专用工具、设备等。

### 4.1.2 被告应诉策略的基本框架

被告的应诉策略应围绕以下框架展开：

**（1）诉因抗辩**

被告应首先分析原告主张的诉因是否成立，包括：诉因对应的行为类型是否构成不正当竞争、原告是否为适格原告、诉讼时效是否届满等。

**（2）证据抗辩**

被告应对原告提交的证据进行严格质证，重点审查：证据的真实性、合法性、关联性；证据的证明力大小；原告举证是否达到证明标准。

**（3）事实抗辩**

被告可以从以下角度进行事实抗辩：否认实施了被诉行为；主张行为具有正当理由；主张原告损失与被告行为之间不存在因果关系等。

**（4）法律抗辩**

被告可以从以下角度进行法律抗辩：主张被诉行为不构成不正当竞争；主张原告主张的法律依据不适用于本案；主张诉讼请求缺乏法律依据等。

### 4.1.3 互联网法院集中管辖新规

#### （1）法释〔2025〕14号：互联网法院管辖新规

**法规信息**：《最高人民法院关于互联网法院案件管辖的规定》，法释〔2025〕14号，2025年10月11日发布，**2025年11月1日起施行**。共4条。

**核心变化——网络不正当竞争纠纷纳入互联网法院集中管辖**：

在2018年规定基础上，法释〔2025〕14号将以下四类新型网络案件新增纳入互联网法院管辖范围：

| 新增案件类型 | 说明 |
|---|---|
| 网络数据权属、侵权、合同纠纷 | 涉及数据确权、数据爬取、数据交易等 |
| 网络个人信息保护、隐私权纠纷 | 涉及个人信息处理中的隐私保护 |
| 网络虚拟财产权属、侵权、合同纠纷 | 涉及虚拟货币、游戏道具等 |
| **网络不正当竞争纠纷** | **对应2025年反法第13条数据专款等** |

**适用范围**：北京市、杭州市、广州市辖区内应由基层法院审理的上述案件，分别由北京互联网法院、杭州互联网法院、广州互联网法院集中管辖。

**协议管辖规则**：就合同及其他财产权益民事纠纷，当事人可依法协议约定与争议有实际联系地点的互联网法院管辖。但格式条款约定管辖的，须符合法律及司法解释关于格式条款的规定（提供方须履行提示与说明义务，否则可能被认定无效或不成为合同内容）。

**涉外案件**：符合上述范围的涉外民事案件以及涉港澳台民事案件，由互联网法院管辖。

**行政案件同步调整**：互联网法院集中管辖所在市辖区内应由基层法院受理的涉及网络数据监管、网络个人信息保护监管、**网络不正当竞争监管**、网络交易管理、网络信息服务管理等行政案件。

**对反法诉讼管辖的实务影响**：

1. **网络不正当竞争案件管辖格局重构**：2025年11月1日后，在北京、杭州、广州三市辖区内，基层法院管辖的网络不正当竞争一审案件统一由互联网法院审理。这意味着大量涉及第13条数据专款、第14条平台强制低价的案件将集中至互联网法院。

2. **管辖选择策略调整**：原告在上述三市提起网络不正当竞争诉讼时，不能再选择普通基层法院，必须向互联网法院起诉。但中级法院以上的级别管辖不受影响。

3. **互联网法院以外的地区**：深圳、成都等其他地区的网络不正当竞争案件仍按原有级别管辖和地域管辖规则，由具有知识产权案件管辖权的基层法院或中级法院审理。

4. **广州互联网法院实践经验**：截至2026年4月，广州互联网法院累计受理涉网络不正当竞争纠纷案件已超1000件，其中约95%为著作权侵权与不正当竞争复合案由案件，积累了丰富的裁判经验。

#### （2）法释〔2025〕15号：协议管辖批复

**法规信息**：《最高人民法院关于部分民事案件管辖适用法律有关问题的批复》，法释〔2025〕15号，2025年12月30日发布，**2025年12月31日起施行**。共5条。

**核心内容**：

1. **"五地"以外协议管辖须有实际联系**：当事人协议选择被告住所地、合同履行地、合同签订地、原告住所地、标的物所在地以外其他地点为管辖连接点的，该地点须与争议有实际联系。无证据证明与争议有实际联系的，管辖协议无效。

2. **协议管辖不得违反专门管辖**：当事人通过管辖协议将法律、司法解释规定只能由专门人民法院管辖的案件约定由其他人民法院管辖的，属于通过管辖协议排除专门管辖，管辖协议无效。**此项规则对反法诉讼的意义**：网络不正当竞争案件依法应由互联网法院集中管辖的，当事人不能通过协议约定由非互联网法院审理。

3. **管辖协议约定不明的处理**：仅约定管辖地域但未约定具体管辖法院的，结合案件性质、标的额等，根据专门管辖、级别管辖等规定能够确定具体管辖法院的，按协议确定管辖。

4. **"或裁或诉"条款中仲裁协议无效不影响管辖协议效力**：仲裁条款无效不必然导致管辖协议无效，两者效力独立判断。

5. **责任保险合同纠纷管辖**：可由被保险人住所地人民法院管辖。

#### （3）法〔2025〕167号：基层法院知产管辖调整

**法规信息**：《最高人民法院关于印发基层人民法院管辖第一审知识产权民事、行政案件有关事项的通知》，法〔2025〕167号，**2025年10月1日起施行**。

该通知对基层法院管辖第一审知识产权民事、行政案件的标准进行了调整，影响反不正当竞争诉讼的级别管辖配置。具体内容为：具有知识产权案件管辖权的基层法院管辖第一审知识产权（含不正当竞争）民事案件的诉讼标的额上限有所调整，超过上限的案件由中级法院管辖。当事人应关注所在地基层法院的最新管辖标准，以正确选择管辖法院。

> **提示**：法〔2025〕167号属授权性框架文件，各地具体诉讼标的额标准以当地高级法院公告为准。建议在立案前查询受诉法院所在省高院发布的最新级别管辖标准。

## 4.2 举证责任分配

### 4.2.1 一般原则

反不正当竞争诉讼适用"谁主张、谁举证"的一般原则。根据法释〔2022〕9号及相关司法解释的规定，原告应就被告实施了不正当竞争行为、原告因此遭受损失、被告行为与损失之间存在因果关系等要件承担举证责任。

**表4-1 反不正当竞争诉讼一般举证责任分配**

| 要件 | 举证方 | 内容 |
|------|-------|------|
| 原告适格 | 原告 | 证明与被告存在竞争关系或竞争利益 |
| 不正当竞争行为 | 原告 | 证明被告实施了被诉行为 |
| 损害后果 | 原告 | 证明因被告行为遭受实际损失 |
| 因果关系 | 原告 | 证明损失与被告行为之间存在因果关系 |
| 被告抗辩事由 | 被告 | 证明存在免责或减责事由 |

### 4.2.2 商业秘密侵权的举证责任移转

2025年修订新增的第三十九条，在商业秘密侵权诉讼中引入了举证责任移转规则，是反法历史上首次引入该制度。

**（1）举证责任移转的适用条件**

根据第三十九条及孔祥俊教授在《反不正当竞争法释论与适用》（2025年）中的解读，举证责任移转的适用须满足以下条件：

| 序号 | 条件 | 说明 |
|------|------|------|
| 1 | 权利人已证明拥有合法商业秘密 | 须证明涉案信息符合秘密性、价值性、保密性三要件 |
| 2 | 权利人已证明涉嫌侵权人有获取渠道 | 如曾是员工、合作伙伴，或有渠道接触相关信息 |
| 3 | 涉嫌侵权人使用的信息实质相同 | 权利人须初步证明两者的信息实质相同 |

满足上述条件后，举证责任转移至涉嫌侵权人，由其证明其不存在侵权行为（如独立研发、反向工程、合法受让等）。

**（2）举证责任移转的适用范围**

孔祥俊教授指出，举证责任移转规则仅适用于民事诉讼程序，行政执法程序不适用该规则。此外，举证责任移转并非"举证责任倒置"——原告仍须首先完成初步举证，证明上述三个条件成立后，方可触发移转效果。

**（3）被告的抗辩事由**

涉嫌侵权人可就以下事由进行抗辩：

1. **独立研发抗辩**：证明相关信息系通过独立研发取得；
2. **反向工程抗辩**：证明相关信息系通过反向工程取得（须注意反向工程须合法且不违反保密义务）；
3. **合法受让抗辩**：证明相关信息系通过合法转让或授权取得；
4. **公知技术抗辩**：证明相关信息属于公知技术，不构成商业秘密。

### 4.2.3 举证责任移转的实践运用

**（1）新能源汽车底盘案中的整体分析法**

在新能源汽车底盘技术秘密案（2024年）中，最高人民法院创立了"举证责任整体分析法"：当被诉侵权人违背研发规律、明显短于独立研发所需时间、有渠道获取技术秘密时，可以推定侵权成立。

这一方法与举证责任移转规则相辅相成——在权利人初步证明侵权人具有获取渠道和时间便利后，法院可以综合整体情况推定侵权成立，无需权利人逐一证明具体的侵权手段。

**（2）举证责任移转与"接触+实质相似"规则的关系**

孔祥俊教授在《反不正当竞争法释论与适用》（2025年）中指出，举证责任移转与"接触+实质相似"规则在功能上有重合，但两者属于不同的制度设计：

| 比较维度 | 举证责任移转（第39条） | "接触+实质相似"规则 |
|---------|----------------------|---------------------|
| 制度来源 | 2025年修订新增 | 司法实践创设 |
| 适用条件 | 权利人证明三要件后移转 | 权利人证明接触+实质相似 |
| 法律效果 | 举证责任移转至被告 | 可推定侵权成立 |
| 关系 | 互补适用 | 互补适用 |

### 4.2.4 电子证据规则

随着数字化程度的加深，电子证据在反不正当竞争诉讼中的重要性日益凸显。2025年，最高人民法院发布了关于在线诉讼中电子证据规则的重要司法解释，对区块链存证、电子证据审查标准等作出了明确规定。

**（1）区块链存证的效力推定**

根据相关规定，经有资质的第三方区块链存证平台存证的电子证据，推定其未经篡改。这一规定对数据不正当竞争案件的证据保全具有重要意义——原告可以通过区块链存证平台对被告的数据抓取行为、平台规则等事实进行及时存证，确保证据的完整性和证明力。

**（2）电子证据的审查标准**

法院对电子证据的审查应综合考量以下因素：

**表4-2 电子证据审查标准**

| 审查因素 | 内容 | 说明 |
|---------|------|------|
| 系统可靠性 | 存证系统的技术可靠性 | 是否采用业界公认的技术标准 |
| 完整性保障 | 哈希值校验等 | 哈希值是否一致，内容是否完整 |
| 内容可展示性 | 电子证据的内容可被展示 | 内容可被正常读取和展示 |
| 身份可确认性 | 电子证据生成者的身份可确认 | 签名、时间戳等身份认证措施 |
| 篡改可能性 | 电子证据是否可能被篡改 | 是否存在被篡改的技术可能性 |
| 关联性 | 电子证据与案件的关联程度 | 内容是否与待证事实相关 |

**（3）举证责任与电子证据**

在涉数据不正当竞争案件中，电子证据的举证往往成为关键。根据相关统计数据，在97件涉数据不正当竞争案件样本中，原告一审胜诉率极高（21/22件获得全部或部分支持），其中一个重要原因在于原告通过电子证据有效证明了被告的数据抓取行为和损害后果。

**表4-3 涉数据不正当竞争案件举证要点**

| 举证事项 | 举证内容 | 证明目的 |
|---------|---------|---------|
| 竞争关系 | 证明原被告之间存在竞争关系或竞争利益 | 证明原告具有诉权 |
| 合法获取 | 证明原告对系争数据享有合法权益 | 证明数据权益归属 |
| 价值存在 | 证明数据具有商业价值 | 证明损害后果的基础 |
| 非法获取 | 证明被告实施了不正当手段获取数据 | 证明侵权行为 |
| 竞争优势 | 证明原告因被告行为丧失竞争优势 | 证明损害后果 |
| 损失大小 | 证明原告因被告行为遭受的实际损失 | 证明赔偿数额 |

## 4.3 被告抗辩策略

### 4.3.1 竞争关系抗辩

**（1）竞争关系的传统理解与突破**

传统理论认为，不正当竞争行为的认定以竞争关系为前提。被告往往以"双方不存在竞争关系"为由进行抗辩。然而，法释〔2022〕9号第二条已将"其他经营者"界定为"与被诉行为人存在可能的争夺交易机会、损害竞争优势关系的经营者"，突破了传统"同业竞争"的狭义理解。

**（2）竞争行为本位论的制度化**

孔祥俊教授在《反不正当竞争法释论与适用》（2025年）中系统论证了"竞争行为本位论"，核心观点包括：

1. **竞争关系非反法适用前提**：竞争关系既非不正当竞争行为的构成要件，更非反法的适用条件；
2. **应从"竞争行为"角度理解不正当竞争**：判断某一行为是否构成不正当竞争，应聚焦于行为本身的性质；
3. **竞争关系仅具程序性意义**：竞争关系在诉讼中的功能主要是认定原告适格地位，而非行为定性的前提条件。

**（3）自媒体、网络大V的特殊地位**

孔祥俊教授进一步指出，在互联网经济中，自媒体、网络大V等主体虽非传统意义上的经营者，但其通过流量变现等方式参与市场竞争，可以成为商业诋毁等行为的适格被告。这一观点为互联网平台上的竞争纠纷处理提供了理论支撑。

**（4）被告的应对策略**

面对原告以竞争关系为由的起诉，被告可以从以下角度进行抗辩：

1. **否认存在竞争关系**：主张双方不属于同一行业，不存在争夺交易机会的关系；
2. **挑战原告的适格地位**：主张原告不是"其他经营者"，不具有诉权；
3. **论证行为不具有竞争属性**：主张被告的行为并非出于竞争目的，不属于反法意义上的竞争行为。

### 4.3.2 不正当性抗辩

被告可以从行为正当性的角度进行抗辩，主张被诉行为不具有"不正当性"。

**（1）自由竞争抗辩**

被告可主张其行为属于自由竞争的范畴，未违反诚实信用原则和商业道德。孔祥俊教授在《反不正当竞争法释论与适用》（2025年）中强调，混淆是划分模仿自由、自由竞争与不正当竞争的根本标准。未导致混淆的模仿行为属于自由竞争范畴，反法不应过度介入。

**（2）"利己不损人"抗辩**

被告可主张其行为属于"利己不损人"的竞争行为，未损害他人合法权益。孔祥俊教授指出，即使竞争行为在道德上值得商榷，但只要不构成对他人合法权益的损害，就不构成反法意义上的不正当竞争。

**（3）公共利益抗辩**

在特定案件中，被告可主张其行为有利于公共利益（如促进技术创新、降低消费者成本等），不构成不正当竞争。

### 4.3.3 商业秘密抗辩的特殊规则

在商业秘密侵权案件中，被告除了一般的抗辩事由外，还享有商业秘密案件特有的抗辩规则。

**（1）公知技术抗辩**

被告可主张涉案信息属于公知技术，不构成商业秘密。根据法释〔2020〕7号的规定，公知技术抗辩的审查应综合考量：相关信息是否为公众普遍知晓、相关信息是否为公众容易获得、相关信息与公知技术的组合是否具有非显而易见性等。

**（2）独立研发抗辩**

被告可主张相关信息系通过独立研发取得，未实施任何不正当手段获取权利人的商业秘密。

**（3）反向工程抗辩**

被告可主张相关信息系通过反向工程取得。根据法释〔2020〕7号第十四条的规定，通过反向工程获得商业秘密的，不构成侵犯商业秘密。但须注意，反向工程须以合法方式实施，且不违反保密义务。

### 4.3.4 损害抗辩

被告可以从损害后果的角度进行抗辩，主张原告主张的损害赔偿缺乏事实依据。

**（1）否认损害存在**

被告可主张原告未遭受实际损失，或损失与被告行为之间不存在因果关系。

**（2）减损抗辩**

被告可主张原告未能采取合理措施减损，对损失扩大部分无权主张赔偿。

**（3）赔偿数额抗辩**

在损害确实存在的情况下，被告可主张原告主张的赔偿数额过高，请求法院根据实际情况酌定。

## 4.4 行为保全

### 4.4.1 行为保全的制度功能

行为保全（诉前禁令、诉中禁令）是反不正当竞争诉讼中的重要程序工具。通过行为保全，法院可以在终审判决前责令被告停止正在实施的不正当竞争行为，及时制止侵权行为的持续和扩大。

根据《民事诉讼法》及法释〔2018〕21号《最高人民法院关于审查知识产权纠纷行为保全案件适用法律若干问题的规定》的规定，行为保全的审查应综合考量以下因素。

### 4.4.2 行为保全的审查标准

**（1）"情况紧急"的认定**

法释〔2018〕21号规定了"情况紧急"的六种认定标准：

| 序号 | 情形 | 说明 |
|------|------|------|
| 1 | 侵权行为正在进行 | 不立即制止将导致损害进一步扩大 |
| 2 | 证据可能灭失 | 不立即保全将导致证据无法取得 |
| 3 | 权利人难以维权 | 不及时保全将导致维权困难 |
| 4 | 行为持续性侵权 | 持续性侵权不制止将造成难以弥补的损害 |
| 5 | 被告可能转移财产 | 转移财产将导致判决难以执行 |
| 6 | 其他紧急情况 | 兜底条款 |

**（2）综合审查四因素**

法院审查行为保全申请，应综合考量以下因素：

1. **胜诉可能性**：申请人主张的权利是否可能得到支持；
2. **损害紧迫性**：不采取行为保全措施是否将导致难以弥补的损害；
3. **利益平衡**：采取行为保全措施对被申请人造成的损害是否大于不采取措施对申请人造成的损害；
4. **公共利益**：采取行为保全措施是否损害公共利益。

**（3）"难以弥补的损害"的认定**

法释〔2018〕21号列举了"难以弥补的损害"的四种典型情形：

| 序号 | 情形 | 说明 |
|------|------|------|
| 1 | 损害具有持续性 | 损害持续扩大，难以用金钱赔偿衡量 |
| 2 | 商誉损害 | 商业信誉一旦受损难以恢复 |
| 3 | 市场机会丧失 | 市场份额的丧失难以逆转 |
| 4 | 其他 | 其他难以用金钱衡量的损害 |

**（4）担保要求与保全期限**

行为保全措施的采取，通常需要申请人提供相应担保。担保的数额应与可能对被申请人造成的损害相当。保全措施的期限应与案件的审理进度相适应，必要时可延长。

### 4.4.3 反不正当竞争案件中行为保全的特殊考量

**（1）商业秘密案件的行为保全**

商业秘密案件中，行为保全具有特殊重要性。由于商业秘密一旦被公开即丧失秘密性，权利人需要及时通过行为保全阻止商业秘密的进一步扩散。同时，行为保全措施的内容应具体明确，包括：停止使用商业秘密、返还或销毁载有商业秘密的文件和设备、禁止向第三方披露等。

**（2）数据竞争案件的行为保全**

在数据不正当竞争案件中，行为保全的重点在于阻止被告继续获取或使用原告的数据。法院在审查数据竞争案件的行为保全申请时，应重点考量：原告对数据是否享有合法权益、被告的数据获取或使用行为是否可能构成不正当竞争、数据的商业价值是否难以用金钱赔偿等。

## 4.5 2025年新法对诉讼策略的影响

### 4.5.1 数据权益诉讼的举证要点调整

2025年修订在第十三条中新增数据保护专款，首次在立法层面确立了数据权益的专门保护规则。这一变化对数据权益诉讼的举证要点产生了深远影响。

**（1）举证框架的变化**

在数据保护专款确立之前，数据权益诉讼主要通过反法一般条款进行，举证框架较为灵活。数据保护专款确立后，原告的举证框架应相应调整：

| 举证事项 | 举证内容 | 变化 |
|---------|---------|------|
| 数据权益 | 证明对系争数据享有合法权益 | 须证明数据为"合法持有的数据" |
| 侵权手段 | 证明被告采用了不正当手段 | 须证明欺诈、胁迫、破坏技术措施等 |
| 损害后果 | 证明损害了合法权益 | 须证明实质性损害 |
| 豁免排除 | 排除豁免情形的适用 | 须证明不属于公开数据等豁免情形 |

**（2）数据权益与商业秘密的关系**

孔祥俊教授在《反不正当竞争法释论与适用》（2025年）中指出，数据权益保护与商业秘密保护路径存在以下关系：

1. **互不排斥**：构成商业秘密的数据可以通过商业秘密保护路径获得保护；不构成商业秘密的数据可以通过数据权益保护路径获得保护；
2. **竞合选择**：原告可根据案件事实选择更有利的保护路径；
3. **举证标准差异**：商业秘密保护路径的举证标准较高，但保护力度较强；数据权益保护路径的举证标准相对灵活。

### 4.5.2 新行为类型拓展的诉讼策略

**（1）平台强制低价诉讼（第十四条）**

第十四条是2025年新增的条款，为平台强制低价行为提供了明确的请求权基础。孔祥俊教授指出，该条款未来可能在民事诉讼中作为起诉依据，但平台责任界限有待司法探索。

**表4-4 平台强制低价诉讼举证要点**

| 举证事项 | 举证内容 |
|---------|---------|
| 主体适格 | 证明被告为平台经营者 |
| 强制行为 | 证明被告利用技术手段或平台规则强制平台内经营者低价销售 |
| 低于成本 | 证明销售价格低于成本 |
| 损害后果 | 证明扰乱了市场竞争秩序 |
| 因果关系 | 证明损害与被告行为之间存在因果关系 |

**（2）滥用优势地位诉讼（第十五条）**

第十五条是2025年另一新增条款，规制具有市场优势地位的经营者滥用其优势地位实施不正当竞争的行为。孔祥俊教授指出，第十五条须适用于特殊领域，指向行业准入限制、行政授权形成的优势地位，而非泛化适用。

**表4-5 滥用优势地位诉讼举证要点**

| 举证事项 | 举证内容 |
|---------|---------|
| 优势地位 | 证明被告在相关市场具有优势地位 |
| 滥用行为 | 证明被告利用其优势地位实施了不正当竞争行为 |
| 损害后果 | 证明扰乱了市场竞争秩序，损害了其他经营者合法权益 |
| 因果关系 | 证明损害与被告行为之间存在因果关系 |

### 4.5.3 域外适用的诉讼策略

2025年修订新增第四十条，明确了反法的域外适用效力：对在中华人民共和国境外实施的不正当竞争行为，扰乱境内市场秩序或者损害境内经营者合法权益的，依照本法处理。

**（1）管辖连接点的确定**

孔祥俊教授在《反不正当竞争法释论与适用》（2025年）中指出，第四十条不是冲突规范，管辖连接点是域外行为对境内市场造成影响。原告在提起域外侵权诉讼时，应重点证明：

1. **域外行为的实施**：被告在境外实施了不正当竞争行为；
2. **境内损害后果**：该行为对境内市场秩序或境内经营者造成了损害；
3. **因果关系**：境内损害与域外行为之间存在因果关系。

**（2）证据跨境调取的挑战**

域外侵权诉讼面临的一大挑战是证据跨境调取的困难。原告可考虑以下策略：

1. **利用国际司法协助**：通过海牙公约等国际条约规定的司法协助渠道调取域外证据；
2. **利用公证认证**：对域外形成的证据进行公证认证，增强其证明力；
3. **利用证据规则**：申请法院要求被告披露相关证据，或通过其他证据规则减轻举证负担。

**（3）法律适用的确定**

在域外侵权诉讼中，法院将依据中国反法确定行为的违法性。同时，法院在认定域外行为时，应综合考量行为地法律、损害结果发生地法律等因素。

### 4.5.4 惩罚性赔偿的诉讼策略

2025年修订进一步强化了惩罚性赔偿制度的适用。在商业秘密侵权案件中，故意侵权且情节严重的，可以在实际损失或侵权获利的1~5倍范围内确定赔偿数额。

**（1）惩罚性赔偿的适用条件**

| 条件 | 说明 |
|------|------|
| 主观要件 | 须为"故意"侵权 |
| 客观要件 | 须"情节严重" |
| 基数确定 | 以实际损失或侵权获利为基数 |
| 倍数确定 | 1~5倍范围内酌定 |

**（2）惩罚性赔偿的举证策略**

原告主张惩罚性赔偿，应重点举证以下事项：

1. **故意的证明**：被告明知涉案信息为商业秘密，仍故意实施侵权行为。具体证据包括：被告曾接触过原告的商业秘密、被告收到过原告的警告函、被告曾因类似行为被处罚等。
2. **情节严重的证明**：侵权行为的规模大、持续时间长、影响范围广等。具体证据包括：侵权产品的销售数量、侵权持续的时间、涉及的地域范围等。

## 4.6 本章小结

本章系统梳理了反不正当竞争诉讼的策略与举证要点，主要结论如下：

**一、举证责任分配**

1. 反不正当竞争诉讼适用"谁主张、谁举证"的一般原则；
2. 2025年修订新增的第三十九条在商业秘密侵权诉讼中引入举证责任移转规则，有效缓解了权利人的举证困难；
3. 电子证据规则日趋完善，区块链存证的效力推定为数据案件的证据保全提供了有力工具。

**二、被告抗辩策略**

1. 竞争行为本位论的确立使竞争关系抗辩的空间缩小；
2. "利己不损人"的行为不属于反法调整范围；
3. 商业秘密案件享有公知技术抗辩、独立研发抗辩、反向工程抗辩等特有抗辩规则。

**三、行为保全**

1. 行为保全的审查标准包括"情况紧急"的认定、综合审查四因素、"难以弥补的损害"的认定；
2. 商业秘密案件和数据竞争案件中，行为保全具有特殊重要性。

**四、2025年新法的影响**

1. 数据保护专款的确立调整了数据权益诉讼的举证框架；
2. 第十四条和第十五条为平台强制低价和滥用优势地位提供了明确的请求权基础；
3. 第四十条域外适用条款为跨境诉讼提供了管辖依据。

# 第五章 法律责任与救济

《反不正当竞争法》第四章（第22—39条）系统规定了不正当竞争行为的法律责任体系，包括民事责任、行政责任和刑事责任三个层次。2025年修订在维持原有责任体系框架的基础上，对处罚力度进行了全面调整，新增了举证责任移转（第三十九条）等程序性规则，并对新增行为类型（平台强制低价、滥用优势地位）配置了相应的法律责任。本章将对各类法律责任的构成要件、计算方式、适用范围进行系统梳理，并分析2025年修订对责任体系的影响。

## 5.1 民事责任概述

### 5.1.1 民事责任的类型体系

《反不正当竞争法》第二十二条规定了不正当竞争行为的民事责任类型体系。根据该条规定，民事责任主要包括以下几种类型：

**（1）停止侵害**

停止侵害是最基本的民事责任形式，适用于正在实施的不正当竞争行为。法院可判令被告停止正在实施的混淆行为、虚假宣传、侵犯商业秘密等行为。根据法释〔2022〕9号第二十五条的规定，混淆行为案件中，法院可依据第六条规定判令被告停止使用相关企业名称。

**（2）消除影响**

消除影响适用于不正当竞争行为给权利人商誉造成损害的情形。法院可判令被告在一定范围内（如特定媒体、特定地域）消除因其不正当竞争行为造成的不良影响。

**（3）赔偿损失**

赔偿损失是民事责任的核心类型，包括补偿性赔偿和惩罚性赔偿两个层次。补偿性赔偿遵循"实际损失→侵权获利→法定赔偿"的递进计算逻辑；惩罚性赔偿则在此基础上叠加，适用于特定情形。

### 5.1.2 赔偿顺序与计算逻辑

《反不正当竞争法》（2025修订）第二十二条规定了民事赔偿的计算顺序：

**表5-1 民事赔偿计算顺序**

| 顺序 | 计算方式 | 适用条件 | 说明 |
|------|---------|---------|------|
| 第一顺位 | 实际损失 | 权利人能够证明 | 按权利人因被侵权所受到的实际损失确定 |
| 第二顺位 | 侵权获利 | 实际损失难以计算 | 按侵权人因侵权所获得的利益确定 |
| 第三顺位 | 法定赔偿 | 上述均难以确定 | 由法院酌定，500万元以下 |
| 叠加适用 | 惩罚性赔偿 | 仅限商业秘密，故意+情节严重 | 在上述基数上叠加1~5倍 |

孔祥俊教授在《反不正当竞争法释论与适用》（2025年）中指出，上述赔偿计算体系遵循"补偿为主、惩罚为辅"的基本原则。补偿性赔偿的目的是使权利人恢复到未受侵害时的状态；惩罚性赔偿则旨在通过超出实际损害的赔偿，遏制恶意侵权行为。

### 5.1.3 合理开支的另行主张

根据第二十二条的规定，赔偿损失与权利人为制止不正当竞争行为所支付的合理开支可以另行主张。这意味着合理开支不计入上述赔偿顺序的计算基数，权利人可以在获得损失赔偿的同时，就合理开支另行主张权利。

合理开支通常包括：律师费、公证费、差旅费、鉴定费等与维权行为直接相关的费用。法院在认定合理开支时，会审查费用的必要性、合理性以及与案件事实的关联性。

## 5.2 损害赔偿计算方式

### 5.2.1 实际损失的计算

实际损失是指权利人因不正当竞争行为直接遭受的经济损失。实际损失的计算通常从以下维度考量：

1. **产品销售收入的减少**：因被诉不正当竞争行为导致的权利人产品或服务销售收入的下降；
2. **利润的减少**：销售收入减少对应的利润损失；
3. **商誉损害**：因不正当竞争行为导致的商业信誉损失，通常以恢复商誉所需的合理费用计算；
4. **其他直接损失**：如为应对不正当竞争行为而支出的额外成本等。

### 5.2.2 侵权获利的计算

当实际损失难以确定时，可按照侵权人因侵权所获得的利益确定赔偿数额。侵权获利的计算通常从以下维度考量：

1. **侵权产品销售收入**：侵权人通过不正当竞争行为获得的销售收入；
2. **侵权产品净利润**：扣除成本后的净利润；
3. **贡献率调整**：如果涉案侵权产品或行为的利润仅部分归因于不正当竞争行为，应按照贡献率进行调整。

孔祥俊教授在《反不正当竞争法释论与适用》（2025年）中指出，侵权获利的计算应注意区分侵权行为贡献与正常商业努力的贡献。在实践中，法院通常会参考技术贡献率、市场贡献率等因素酌定贡献率。

### 5.2.3 法定赔偿

当实际损失和侵权获利均难以确定时，由法院根据侵权行为的情节判决给予500万元以下的法定赔偿。法定赔偿的适用具有以下特点：

1. **兜底性**：法定赔偿是赔偿计算的最后兜底方式，只有在实际损失和侵权获利均无法确定时方可适用；
2. **酌定性**：法定赔偿数额由法院综合考量侵权行为的性质、持续时间、影响范围、侵权人的主观过错等因素酌定；
3. **上限控制**：法定赔偿设有500万元的上限。

**表5-2 损害赔偿计算方式对比**

| 计算方式 | 适用条件 | 计算基准 | 备注 |
|---------|---------|---------|------|
| 实际损失 | 权利人能够证明 | 权利人因侵权减少的收入和利润 | 举证难度较高 |
| 侵权获利 | 实际损失难以计算 | 侵权人因侵权的获利 | 可能与实际损失存在差异 |
| 法定赔偿 | 上述均难以确定 | 法院酌定 | 上限500万元 |
| 惩罚性赔偿 | 仅限商业秘密，故意+情节严重 | 上述基数×1~5倍 | 可与上述任一叠加 |

## 5.3 惩罚性赔偿

### 5.3.1 惩罚性赔偿的适用范围

根据《反不正当竞争法》（2025修订）第二十二条的规定，惩罚性赔偿仅适用于侵犯商业秘密的情形。这是反法中唯一的惩罚性赔偿条款，体现了立法者对商业秘密保护的特别重视。

孔祥俊教授在《反不正当竞争法释论与适用》（2025年）中指出，将惩罚性赔偿的适用范围限定为商业秘密领域，是经过审慎考量的制度选择。商业秘密不同于专利、商标等注册型知识产权，其保护状态不具有公示性，第三方的注意成本较高，因此通过惩罚性赔偿增强威慑力具有正当性。

### 5.3.2 惩罚性赔偿的适用条件

惩罚性赔偿的适用须同时满足以下条件：

**（1）主观要件：故意侵权**

根据法释〔2020〕7号第三条的规定，"故意"的认定应综合考量以下因素：侵权人是否明知涉案信息为商业秘密；侵权人是否有获取涉案商业秘密的渠道或机会；侵权人是否收到过权利人的警告函或律师函；侵权人是否曾因类似行为受到行政处罚或刑事追诉等。

**（2）客观要件：情节严重**

根据法释〔2020〕7号第四条的规定，"情节严重"的认定应综合考量以下因素：侵权行为的规模（如侵权产品的数量、销售金额）；侵权行为的持续时间；侵权行为的地域范围；侵权行为对权利人造成的损害程度；侵权人对侵权获利的态度（如持续侵权、拒不履行判决等）。

**（3）基数要件：存在补偿性赔偿的计算基数**

惩罚性赔偿须在补偿性赔偿基数上叠加适用，基数可以是实际损失、侵权获利或法定赔偿中的任一确定数额。

**表5-3 惩罚性赔偿适用要件**

| 要件 | 内容 | 说明 |
|------|------|------|
| 主观要件 | 故意 | 明知为商业秘密仍故意侵权 |
| 客观要件 | 情节严重 | 规模大、持续时间长、影响范围广等 |
| 基数要件 | 补偿性赔偿基数 | 实际损失/侵权获利/法定赔偿任一 |

### 5.3.3 惩罚性赔偿的倍数确定

惩罚性赔偿的倍数在1~5倍范围内确定。根据法释〔2020〕7号第八条、第九条的规定，法院在确定具体倍数时应综合考量以下因素：

1. **侵权人的主观过错程度**：故意越明显，倍数越高；
2. **侵权行为的情节严重程度**：情节越严重，倍数越高；
3. **侵权获利情况**：侵权获利越多，倍数可能越高；
4. **权利人的维权成本**：维权成本越高，倍数可能越高；
5. **社会影响**：侵权行为的社会影响越大，倍数可能越高。

### 5.3.4 典型案例：新能源汽车底盘案6.4亿余元惩罚性赔偿

在新能源汽车底盘技术秘密案（2024年）中，最高人民法院二审判决确定惩罚性赔偿2倍，最终赔偿金额达6.4亿余元，是迄今为止商业秘密惩罚性赔偿金额最高的判决。

**裁判要点**：

1. **故意的认定**：侵权人有组织、有计划以不正当手段大规模挖取技术人才及资源，具有明显的侵权故意；
2. **情节严重的认定**：侵权规模大（涉案技术人员约40人）、持续时间长、影响范围广；
3. **基数的确定**：以侵权获利为基数，结合利润贡献率（8%）确定；
4. **倍数的确定**：综合考量主观过错和客观情节，确定2倍惩罚性赔偿。

孔祥俊教授在《反不正当竞争法释论与适用》（2025年）中高度评价本案的示范意义：6.4亿余元的惩罚性赔偿充分体现了对恶意侵权行为的严厉惩处，对于遏制商业秘密侵权行为具有重要的震慑效果。

### 5.4.1 行政责任体系概述

《反不正当竞争法》第四章对各类不正当竞争行为规定了行政责任，由市场监督管理部门负责查处。2025年修订对行政处罚的数额幅度进行了全面调整，总体呈现处罚力度加大的趋势。

### 5.4.2 行政处罚的适用规则

**（1）处罚主体的确定**

根据《反不正当竞争法》第五条的规定，监督检查部门依照本法规定负责对不正当竞争行为进行监督检查。县级以上人民政府市场监督管理部门是主要的执法机关。

**（2）处罚幅度的裁量**

市场监督管理部门在确定具体处罚数额时，应综合考量以下因素：违法行为的性质、持续时间、危害后果；违法经营额的大小；当事人的主观过错程度；当事人是否积极整改、消除影响等。

**（3）从轻、减轻处罚情形**

2025年修订新增了从轻或减轻处罚的情形。根据第三十五条的规定，当事人有下列情形之一的，可以依法从轻或者减轻行政处罚：主动消除或者减轻违法行为危害后果的；受他人胁迫或者诱骗实施违法行为的；主动供述市场监督管理部门尚未掌握的违法行为的；积极配合市场监督管理部门查处违法行为并有立功表现的。

### 5.4.3 信用记录与公示

2025年修订新增了信用记录公示制度。根据第三十三条的规定，市场监督管理部门应当将行政处罚信息记入信用记录，并依法通过国家企业信用信息公示系统向社会公示。信用记录的建立，对于强化不正当竞争行为的社会惩戒效果具有重要意义。

孔祥俊教授在《反不正当竞争法释论与适用》（2025年）中指出，信用记录公示制度是"一处违法、处处受限"监管理念的制度化表达，有助于通过社会信用体系的整体约束力，增强对不正当竞争行为的遏制效果。

## 5.5 刑事责任与行刑衔接

### 5.5.1 刑事责任概述

不正当竞争行为在严重情况下可能构成犯罪，须依法追究刑事责任。《反不正当竞争法》第三十八条规定，违反本法规定，构成犯罪的，依法追究刑事责任。

### 5.5.3 典型刑事案例

**案例一：四川凉山"网红直播带货"虚假广告案**

**案号**：刑事案件

**裁判要点**：网络主播在直播带货过程中，对所销售商品的质量、性能、功能等进行虚假宣传，情节严重，构成虚假广告罪。主犯被判处有期徒刑3年2个月。

**案例二："发之萃"生发产品虚假宣传案**

**裁判要点**：经营者对所销售产品进行虚假宣传，且在销售过程中具有非法占有消费者财物的目的，法院认定构成诈骗罪而非虚假广告罪。本案体现了"具有非法占有目的"作为诈骗罪构成要件的关键意义。

**案例三："淘车大师"非法获取计算机信息系统数据案**

**裁判要点**：行为人未经授权进入他人计算机系统，获取车辆评估数据等计算机信息系统数据，情节严重，构成非法获取计算机信息系统数据罪。本案说明数据抓取行为可能同时触发行政责任和刑事责任。

**案例四：四川"川金象"三聚氰胺技术泄密案**

**裁判要点**：涉案技术人员窃取权利人的三聚氰胺技术秘密并提供给第三方使用，情节严重，构成侵犯商业秘密罪。被告人被判处有期徒刑5年6个月，并处罚金280万元。

孔祥俊教授在《反不正当竞争法释论与适用》（2025年）中指出，不正当竞争行为的行刑衔接应注意以下问题：

1. **构成要件的差异**：行政违法与刑事犯罪的构成要件存在质的差异，轻微的行政违法行为不必然构成犯罪；
2. **证据标准的差异**：刑事诉讼的证明标准（排除合理怀疑）高于民事诉讼和行政执法的证明标准（优势证据）；
3. **程序保障的差异**：刑事诉讼享有更为严格的程序保障，侦查、起诉、审判各环节相互制约。

## 5.6 救济途径

### 5.6.1 行政复议

根据《反不正当竞争法》第三十六条的规定，当事人对市场监督管理部门作出的行政处罚决定不服的，可以依法申请行政复议。行政复议是行政相对人寻求救济的第一道程序，通常由本级人民政府或上一级主管部门受理。

行政复议的审查范围包括：行政处罚决定的事实认定是否清楚、证据是否确凿；适用法律是否正确；程序是否合法；处罚是否适当等。

### 5.6.2 行政诉讼

当事人对行政复议决定不服的，或者复议机关逾期不作出复议决定的，当事人可以依法提起行政诉讼。行政诉讼由人民法院管辖，对行政处罚决定的合法性进行全面审查。

根据法释〔2022〕9号第二十六条的规定，反不正当竞争案件的管辖法院为侵权行为地或者被告住所地人民法院。侵权行为地包括侵权行为实施地和侵权结果发生地。

### 5.6.3 诉讼时效

反不正当竞争民事诉讼适用《民法典》关于诉讼时效的一般规定。根据《民法典》第一百八十八条的规定，向人民法院请求保护民事权利的诉讼时效期间为三年，从权利人知道或者应当知道权利受到损害以及义务人之日起计算。

法释〔2022〕9号第二十八条还规定了新旧法适用的过渡规则：人民法院应当依据行为发生时的法律规定认定被诉行为的违法性；但新法的规定不认为违法或者处罚较轻的，适用新法的规定。

### 5.6.4 民事责任与行政责任、刑事责任的协调

《反不正当竞争法》第三十四条确立了民事责任优先原则：经营者财产不足以同时承担民事赔偿责任和缴纳罚款、罚金时，先承担民事赔偿责任。

这一规定体现了对受害者私权救济的优先保障，确保行政处罚和刑事罚金的执行不会损害受害人的民事获偿权。

## 5.7 2025年修订对责任体系的影响

### 5.7.1 处罚力度的全面提升

2025年修订对行政处罚的数额幅度进行了全面调整，整体呈现处罚力度加大的趋势：

**表5-6 2025年修订行政处罚调整要点**

| 行为类型 | 调整前 | 调整后 | 调整幅度 |
|---------|-------|-------|---------|
| 商业贿赂（单位行贿） | 最高300万元 | 最高500万元 | 约67% |
| 商业贿赂（个人） | 未规定 | 最高100万元（行贿）/50万元（受贿） | 新增个人责任 |
| 虚假宣传 | 100万元以下 | 200万元以下（情节严重） | 提高上限 |
| 平台强制低价 | 未规定 | 最高200万元（情节严重） | 新增 |
| 滥用优势地位 | 未规定 | 最高500万元（情节严重） | 新增 |

孔祥俊教授在《反不正当竞争法释论与适用》（2025年）中指出，处罚力度的提升体现了立法者对不正当竞争行为"零容忍"的鲜明态度，有助于形成有效的法律震慑。

### 5.7.2 新增责任类型的配置

2025年修订新增了两类不正当竞争行为，并为其配置了相应的法律责任：

**（1）平台强制低价（第三十条）**

平台强制低价行为的行政处罚分为两档：一般情形处5万元以上50万元以下罚款；情节严重的处50万元以上200万元以下罚款。

**（2）滥用优势地位（第三十一条）**

滥用优势地位行为的行政处罚采用"先责令改正、后罚款"的双层结构：一般情形先责令限期改正，逾期不改正的处100万元以下罚款；情节严重的处100万元以上500万元以下罚款。

### 5.7.3 举证责任移转的程序性影响

2025年修订新增的第三十九条确立了商业秘密侵权诉讼中的举证责任移转规则，对诉讼程序产生了深远影响：

1. **原告初步举证的简化**：原告须首先证明其拥有合法商业秘密、侵权人有获取渠道、使用信息实质相同，此后举证责任移转至被告；
2. **被告抗辩负担的加重**：被告须证明其不存在侵权行为（如独立研发、反向工程等），否则承担不利后果；
3. **裁判效率的提升**：举证责任移转有助于法院在证据掌握相对充分的条件下进行裁判，提升裁判质量和效率。

## 5.8 本章小结

本章系统梳理了《反不正当竞争法》规定的法律责任体系，主要结论如下：

**一、民事责任**

1. 赔偿顺序为"实际损失→侵权获利→法定赔偿（500万元以下）"；
2. 惩罚性赔偿仅适用于商业秘密领域，须满足"故意+情节严重"要件，倍数为1~5倍；
3. 新能源汽车底盘案6.4亿余元惩罚性赔偿创下商业秘密案件赔偿纪录。

**二、行政责任**

1. 2025年修订全面提升了行政处罚力度，商业贿赂最高罚款从300万元提升至500万元；
2. 新增平台强制低价、滥用优势地位两类行为的行政处罚规定；
3. 信用记录公示制度增强了社会惩戒效果。

**三、刑事责任**

1. 不正当竞争行为可能与虚假广告罪、诈骗罪、非法获取计算机信息系统数据罪、侵犯商业秘密罪等罪名衔接；
2. 刑事犯罪的构成要件与行政违法存在质的差异，须严格区分；
3. 民事责任优先于行政罚款和刑事罚金。

**四、救济途径**

1. 当事人可通过行政复议、行政诉讼寻求救济；
2. 民事诉讼适用3年诉讼时效；
3. 新旧法适用的过渡规则以行为发生时间为准。

# 第六章 新兴领域反不正当竞争前沿问题

数字经济的纵深发展、人工智能技术的快速迭代以及国际竞争格局的深刻变革，将反不正当竞争法推至法律体系的前沿地带。2025年《反不正当竞争法》修订回应了平台经济、数据竞争等核心议题，而最高人民法院第47批指导性案例（262-267号）以及德国UWG 2026年修订则进一步拓展了理论视野。本章系统梳理数据权益竞争、平台经济治理、商业秘密保护新发展、德国UWG 2026年修订等前沿议题，以期为实务工作提供参考，并为未来制度完善提供方向性思考。

| 案例号 | 核心规则 | 与新法第13条衔接 |
|--------|---------|-----------------|
| 262号 | 实质性替代三要素（未经许可获取+实质性替代+三元利益损害） | 为"损害合法权益"认定提供参考 |
| 263号 | 用户授权数据转移不构成不正当竞争 | 为"不正当方式"认定提供反向边界 |
| 264号 | 公开数据自由流动原则 | 为"合法持有"数据的合理使用提供依据 |
| 265号 | 个人信息保护与数据竞争的平衡 | 涉及个人信息的数据抓取应兼顾保护与流通 |
| 266号 | 个人信息主体权利的优先性 | 数据权益不得对抗更高位阶权益 |
| 267号 | 账号执行问题 | 裁判执行层面的操作规则 |

### 6.1.2 实质性替代的司法认定

**（1）实质性替代的"三进三出"立法史**

"实质性替代"是数据竞争案件中最核心的损害认定标准，其立法历程颇具戏剧性：

| 时间 | 文件 | 结果 |
|------|------|------|
| 2021年8月 | 《禁止网络不正当竞争行为规定（征求意见稿）》 | 一进（首次引入） |
| 2024年5月 | 《网络反不正当竞争暂行规定》第19条 | 继续保留 |
| 2022年3月 | 《反法司法解释》正式版 | 完全删除（两出） |
| 2025年6月 | 《反不正当竞争法》正式修订版 | 未纳入（三出） |

虽然"实质性替代"未能进入2025年修订的正式条文，但孔祥俊教授在《反不正当竞争法释论与适用》（2025年）中指出，实质性替代要件仍可作为"损害其他经营者合法权益"的定量考量因素，在司法实践中继续发挥作用。

**（2）实质性替代的三要素**

根据第262号指导性案例的裁判规则，实质性替代的认定须同时满足以下三要素：

1. **未经许可获取**：数据获取未获得数据权利人的许可；
2. **实质性替代**：被诉行为对数据的使用对权利人的产品或服务形成了实质性替代；
3. **三元利益损害**：该替代行为损害了权利人、消费者或公共利益中的至少一者。

### 6.1.3 数据权益保护的边界

**（1）用户授权数据转移的反向边界**

第263号指导性案例明确了用户授权数据转移的正当性边界：当数据迁移是基于用户明确授权，且数据转移未损害数据原始控制者的合法权益时，不构成不正当竞争。这一规则为数据迁移实践提供了明确的合规指引。

**（2）公开数据自由流动原则**

第264号指导性案例确立了公开数据自由流动原则：通过合法、公开渠道可以获取的数据，其获取和利用行为原则上不构成不正当竞争。孔祥俊教授在《反不正当竞争法释论与适用》（2025年）中指出，这一原则为数据要素流通提供了制度空间，避免了数据权益保护的过度扩张。

**（3）个人信息保护与数据竞争的平衡**

第265号、第266号指导性案例关注了个人信息保护与数据竞争的交叉地带：涉及个人信息的数据抓取行为，应当兼顾个人信息保护和数据的流通利用。具体而言：

1. 个人信息主体权利具有优先性，数据权益不得对抗更高位阶的个人信息权益；
2. 数据抓取行为不得侵害个人信息主体的知情权、选择权等法定权利；
3. 在保护与流通之间寻求平衡，是数据竞争法规制的核心命题。

### 6.1.4 数据保护专款的适用逻辑

2025年修订新增的第十三条第二款数据保护专款，为数据不正当竞争提供了明确的请求权基础。孔祥俊教授在《反不正当竞争法释论与适用》（2025年）中系统论述了数据保护专款的适用逻辑：

**（1）构成要件**

数据保护专款的适用须同时满足：
1. **不正当手段**：欺诈、胁迫、避开或破坏技术管理措施等；
2. **受保护数据的适格性**：须为"合法持有的数据"；
3. **损害后果**：损害其他经营者的合法权益，扰乱市场竞争秩序。

**（2）豁免情形**

数据保护专款设有以下豁免情形，为正当的数据流通预留了制度空间：
1. 可从公开渠道自由获取的数据；
2. 使用数量不足以造成实质性损害；
3. 不与权利人正常经营冲突；
4. 以诚信方式使用的其他情形。

## 6.2 平台经济竞争治理

### 6.2.1 平台强制低价（第十四条）

**（1）条款背景**

第十四条是2025年修订新增条款，旨在规制平台经济中日益突出的强制低价问题。近年来，部分大型平台利用其优势地位，强制平台内经营者以低于成本的价格销售商品，严重扰乱了正常的市场竞争秩序。

**（2）构成要件**

**表6-2 平台强制低价构成要件**

| 要件 | 内容 | 说明 |
|------|------|------|
| 主体 | 平台经营者 | 利用网络为平台内经营者提供交易场所或服务的经营者 |
| 手段 | 技术手段、平台规则等方式 | 包括算法推荐、搜索降权、扣减保证金、限制流量等 |
| 行为 | 强制平台内经营者以低于成本价格销售 | "强制"强调行为的不合理性，"低于成本"强调经济损害性 |
| 结果 | 扰乱市场竞争秩序 | 须对市场竞争秩序造成损害 |

**（3）与相关制度的区分**

第十四条与《电子商务法》第三十五条存在规制交叉，但侧重点不同：《电子商务法》侧重于保护平台内经营者的自主经营权；《反不正当竞争法》侧重于维护市场竞争秩序。两者的制度功能互补而非替代。

### 6.2.2 滥用优势地位（第十五条）

**（1）条款背景**

第十五条是2025年修订另一新增条款，规制具有市场优势地位的经营者滥用其优势地位实施不正当竞争的行为。孔祥俊教授在《反不正当竞争法释论与适用》（2025年）中指出，第十五条须适用于特殊领域，主要指向行业准入限制、行政授权形成的优势地位，而非泛化适用。

**（2）与《反垄断法》的区分**

**表6-3 滥用优势地位与滥用市场支配地位的区分**

| 比较维度 | 第十五条（滥用优势地位） | 《反垄断法》（滥用市场支配地位） |
|---------|----------------------|----------------------------|
| 认定门槛 | 较低（"优势地位"） | 较高（"支配地位"） |
| 适用标准 | 相对灵活 | 严格（须界定相关市场） |
| 举证责任 | 原告举证难度较低 | 原告举证难度较高 |
| 处罚力度 | 最高500万元（反法标准） | 上一年度销售额1%~10% |
| 制度定位 | 补充《反垄断法》 | 独立规制体系 |

**（3）滥用优势地位与大型企业拖欠账款**

市场监管总局在2026年执法指南中明确将"大型企业拖欠账款"列为滥用优势地位的重点整治领域。大型企业利用其市场优势地位，强制平台内经营者或供应商接受不合理交易条件，拖延付款等行为，可能触发第十五条的规制。

### 6.2.3 平台经济的综合治理框架

孔祥俊教授在《反不正当竞争法释论与适用》（2025年）中指出，平台经济的竞争治理需要多维度、多层次的综合施策：

1. **法律层面的规制**：第十四条、第十五条与《电子商务法》第三十五条形成互补的法律规制框架；
2. **行政层面的监管**：市场监管部门对平台经济的专项执法，与网信部门、工信部门的协同监管；
3. **行业层面的自律**：平台经营者建立健全内部合规机制，强化平台生态的自我净化；
4. **社会层面的监督**：鼓励社会公众对平台不正当竞争行为的举报和监督。

## 6.3 "内卷式"竞争整治

### 6.3.1 市场监管总局2026年执法重点

2026年3月30日，国家市场监督管理总局发布年度执法指南，将"内卷式"竞争整治列为重点工作领域。

**表6-4 2026年五大重点执法领域**

| 领域 | 重点内容 |
|------|---------|
| **"内卷式"竞争整治** | 平台经济，光伏、锂电池、新能源汽车；强制低于成本销售 |
| **大型企业拖欠账款** | 滥用优势地位，强制接受不合理交易条件 |
| **网络不正当竞争** | AI等新兴产业；流量劫持、恶意干扰、滥用平台规则 |
| **传统不正当竞争** | 仿冒混淆、虚假宣传、侵犯商业秘密 |
| **商业秘密保护** | 推进国推商业秘密保护管理体系认证 |

### 6.3.2 "内卷式"竞争的法律界定

"内卷式"竞争并非法律概念，而是对特定竞争乱象的描述性概括。从法律角度看，"内卷式"竞争主要表现为以下类型：

1. **低于成本倾销**：在特定领域（如光伏、锂电池、新能源汽车）实施低于成本的倾销，意图挤垮竞争对手；
2. **强制捆绑销售**：利用优势地位强制交易相对方接受不合理交易条件；
3. **恶意压价竞争**：通过恶意压低价格争夺市场份额，扰乱正常市场秩序。

上述行为可能分别触发第十四条（平台强制低价）、第十五条（滥用优势地位）的规制。

### 6.3.3 新能源汽车行业的特殊性

新能源汽车行业是"内卷式"竞争整治的重点领域。该行业的特殊性在于：

1. **政策驱动与市场竞争并行**：新能源汽车产业享受国家政策扶持，市场竞争与产业政策交织；
2. **技术迭代快、价格战激烈**：新技术快速迭代，企业通过价格战争夺市场份额；
3. **平台经济深度参与**：新能源汽车的销售越来越多地通过电商平台进行。

孔祥俊教授在《反不正当竞争法释论与适用》（2025年）中指出，对新能源汽车等战略性新兴产业的竞争治理，需要在维护竞争秩序与鼓励创新发展之间寻求平衡。

## 6.4 新兴领域不正当竞争

### 6.4.1 人工智能与反不正当竞争

人工智能技术的快速发展催生了新型的不正当竞争形态，主要体现在以下几个方面：

**（1）AI模型参数与结构的保护**

在"变身漫画特效"案（2023年京73民终3802号）中，法院明确认定人工智能模型的参数和结构属于技术秘密范畴，受反不正当竞争法保护。这一裁判为AI领域的竞争规则提供了重要指引。

**（2）AI生成内容的不正当竞争风险**

利用AI批量生成虚假评价、虚假宣传内容的行为，可能触发第九条（虚假宣传）的规制。2025年修订新增的虚假评价条款，明确将"利用虚假或者他人的名义进行虚假评价"纳入规制范围，为AI生成虚假内容的规制提供了明确依据。

**（3）AI竞争的法律边界**

孔祥俊教授在《反不正当竞争法释论与适用》（2025年）中指出，AI领域的竞争法律边界主要包括：

1. **数据获取的正当性**：AI训练数据的获取是否通过正当途径，是否侵犯他人数据权益；
2. **模型输出的竞争属性**：AI模型输出是否构成对他人商业成果的不正当利用；
3. **算法透明与商业秘密的平衡**：算法透明要求与商业秘密保护之间的张力。

### 6.4.2 流量劫持与恶意干扰

流量劫持、恶意干扰是网络不正当竞争的常见形态，主要表现为：

1. **DNS劫持**：通过篡改DNS解析结果，将用户引导至其他网站；
2. **流量劫持**：通过技术手段劫持用户流量，截取本应属于他人的网络访问；
3. **恶意干扰**：通过技术手段干扰他人网站的正常运行。

上述行为可能触发第十三条（互联网不正当竞争）的规制。根据第十三条的规定，强制跳转、误导卸载、恶意不兼容等行为均构成不正当竞争。

### 6.4.3 直播电商与社交媒体竞争

直播电商和社交媒体领域的竞争呈现出以下新特点：

**（1）引流行为的竞争属性**

在"引流直播带货"案（2025年浙08民终563号）中，法院认定短视频引流行为构成仿冒混淆。利用他人注册商标进行引流，导致消费者误以为与权利人存在特定联系的，构成混淆行为。

**（2）虚假宣传的新形态**

直播电商中的虚假宣传呈现出新的特点：主播夸大产品功效、虚构用户评价、虚构销售数据等行为，均可能触发第九条（虚假宣传）的规制。

**（3）商业诋毁的传播效应**

社交媒体上商业诋毁的传播速度和范围远超传统媒介。一条负面评价可能在短时间内被大量转发，造成难以挽回的商誉损失。第十二条（损害商业信誉）的规制范围涵盖社交媒体上的商业诋毁行为。

## 6.5 商业秘密保护新发展

### 6.5.1 举证责任移转的实践影响

2025年修订新增的第三十九条，在商业秘密侵权诉讼中引入了举证责任移转规则，是商业秘密保护领域的重大制度创新。

**（1）适用条件**

举证责任移转须满足以下条件：
1. 权利人已证明其拥有合法的商业秘密；
2. 权利人已证明涉嫌侵权人有渠道或机会获取该商业秘密；
3. 涉嫌侵权人使用的信息与权利人的商业秘密实质相同。

**（2）对司法实践的影响**

孔祥俊教授在《反不正当竞争法释论与适用》（2025年）中指出，举证责任移转规则对司法实践产生了以下影响：

1. **降低维权门槛**：原告的初步举证负担减轻，有利于激发权利人的维权意愿；
2. **强化程序保障**：被告须积极举证证明其不存在侵权行为，否则承担不利后果；
3. **提升裁判质量**：在证据掌握相对充分的条件下裁判，有助于提升裁判的准确性和公信力。

### 6.5.2 国推商业秘密保护管理体系认证

市场监管总局在2026年执法指南中将"推进国推商业秘密保护管理体系认证"列为重点工作。商业秘密保护管理体系认证是指企业依据国家标准（GB/T 29490-2023《企业商业秘密管理规范》）建立的商业秘密保护体系，通过第三方认证机构的审核后获得的认证证书。

**（1）认证的价值**

1. **证明企业已建立完善的商业秘密保护体系**；
2. **在侵权诉讼中，认证证书可作为保密措施合理性的证据**；
3. **提升企业在商业往来中的商业秘密保护信誉**。

**（2）认证与举证责任移转的衔接**

获得国推认证的企业，在主张商业秘密侵权时，可将认证作为保密措施合理性的初步证据，减轻举证负担。

### 6.5.3 新能源汽车底盘案的示范意义

新能源汽车底盘技术秘密案（6.4亿余元惩罚性赔偿）是商业秘密保护的里程碑案件，其示范意义体现在：

1. **举证责任整体分析法**：通过综合考量研发时间、渠道便利性等因素推定侵权，有效缓解举证困难；
2. **迟延履行金制度**：违反停止使用义务每日100万元，擅自处分专利每件100万元；
3. **惩罚性赔偿的威慑力**：6.4亿余元的赔偿充分体现了对恶意侵权的严厉惩处。

## 6.6 德国UWG 2026年修订

### 6.6.1 修订背景与核心框架

德国《反不正当竞争法》（UWG）于2026年9月27日完成修订并生效。此次修订的核心目的是转化欧盟《赋能消费者绿色转型指令》（Empowering Consumers Directive）中关于绿色声明和数字化产品的新规定，标志着德国反不正当竞争立法的最新发展。

**表6-5 德国UWG 2026年修订要点**

| 修订领域 | 主要内容 |
|---------|---------|
| **新附录第23d条** | 七类被禁止的数字化产品行为 |
| **第4a条** | 环境声明新要求（"环保""气候中和"须符合公认标准） |
| **第2a条** | 未经独立认证的可持续性标识违法 |

| 3 | 计划性报废 | 通过设计使产品在一定时间后性能下降，诱导消费者更换 |
| 4 | 虚假耐用性声明 | 对产品实际使用寿命或耐久性作出虚假声明 |
| 5 | 虚假可修复性声明 | 声称产品可修复但实际无法修复或修复成本过高 |
| 6 | 诱导过早更换耗材 | 诱导消费者在耗材尚可使用时更换 |
| 7 | 隐瞒第三方配件兼容性 | 隐瞒产品与第三方配件的不兼容性 |

### 6.6.3 环境声明的新规制

**（1）第4a条：环境声明的准确性要求**

第4a条规定，"环保""气候中和""可持续"等环境声明须符合公认标准，且须有充分的科学依据支撑。这一规定直接针对实践中普遍存在的"漂绿"（greenwashing）行为。

**（2）第2a条：可持续性标识的认证要求**

未经独立认证的可持续性标识不得使用。这意味着企业如需在产品或服务上使用"可持续""环保"等标识，须通过第三方机构的独立认证。

**（3）第4c条：碳中和声明的限制**

仅基于碳抵消（如购买碳信用）的碳中和声明被明确禁止。真正的碳中和须通过减少碳排放和碳捕集等实质性措施实现，而非单纯依赖碳抵消。

### 6.7.3 孔祥俊理论视角下的2025年修法评析

> **说明**：本节从孔祥俊教授的理论视角评析2025年修法，将各章分散引用的孔祥俊理论观点进行系统整合，但不替代各章原有引用——各章节正文中的孔祥俊理论引用系各章节有机组成部分，予以保留。

#### （1）一般条款谦抑性

孔祥俊教授提出反法一般条款（第2条）的适用经历了"由封闭→开放→谦抑"的演进。2025年修法新增第13-15条等具体行为类型，进一步压缩了一般条款的适用空间——对于已有具体条款规制的情形，不得再适用一般条款。这体现了"具体条款优先、一般条款兜底"的适用逻辑。

#### （2）竞争行为本位论

孔祥俊教授主张以"竞争行为"而非"竞争关系"作为反法规制的核心要素。2025年反法第13条对数据不正当竞争行为的规制不限于同业竞争者，任何利用数据、算法实施不正当竞争行为的主体均可成为被告，体现了竞争行为本位论的立法取向。

#### （3）商业道德双标准

孔祥俊教授提出商业道德判断应采用"行业惯例经验标准"+"市场竞争效果功能标准"的双轨框架。2025年反法新增条款对平台强制低价（第14条）和滥用优势地位（第15条）的规制，实质上是从市场竞争效果功能标准出发，对平台经济中的"商业道德"进行了立法层面的界定。

#### （4）数据保护"类权利和弱权利"定位

孔祥俊教授将数据保护定位为"类权利和弱权利"——通过行为规制（而非绝对权）保护数据权益。第13条数据专款的行为规制路径印证了这一定位：法律不赋予数据持有者排他性权利，而是禁止不正当的获取和使用行为，为数据流通预留制度空间。

## 6.8 本章小结

本章系统梳理了反不正当竞争领域的前沿问题，主要结论如下：

**一、数据权益竞争**

1. 最高人民法院第47批指导性案例（262-267号）为数据权益保护提供了系统性指引，确立了实质性替代三要素、公开数据自由流动原则等核心规则；
2. 虽然"实质性替代"未能进入2025年修订的正式条文，但仍可作为"损害合法权益"的定量考量因素；
3. 第十三条数据保护专款为数据不正当竞争提供了明确的请求权基础。

**二、平台经济治理**

1. 第十四条、第十五条为平台强制低价和滥用优势地位提供了明确的规制依据；
2. 两条款与《电子商务法》第三十五条形成互补，与《反垄断法》形成梯度衔接；
3. 平台经济的竞争治理需要法律、行政、行业、社会多维度协同。

**三、新兴领域**

1. 人工智能带来AI模型保护、生成内容规制等新型问题；
2. "内卷式"竞争整治聚焦平台经济、光伏、锂电池、新能源汽车等领域；
3. 直播电商和社交媒体领域的不正当竞争呈现新特点。

**四、商业秘密保护**

1. 第三十九条举证责任移转规则有效降低维权门槛；
2. 国推商业秘密保护管理体系认证有助于提升保护规范化水平；
3. 新能源汽车底盘案确立的迟延履行金制度具有重要示范意义。

**五、中德比较**

1. 德国UWG 2026年修订聚焦数字化产品和环境声明的新规制；
2. 中国2025年修订聚焦数据竞争和平台经济；
3. 两国立法呈现不同发展重点，可相互借鉴。

| 总体原则 | 第1条 | **不得以长期、大额补贴排除限制竞争、扰乱竞争秩序** |
| 具体要求 | 第2条 | 不得强制平台内经营者参加补贴活动或承担补贴成本 |
| | 第3条 | 应利用自有经营利润等合法合规资金实施补贴，不得利用资本优势实施垄断或不正当竞争 |
| | 第4条 | 不得导致平台内经营者以低于成本价格销售商品，扰乱价格秩序 |
| | 第5条 | 坚持公平、合理、无歧视原则，公平对待经营者、骑手、消费者 |
| 监督措施 | 第6-7条 | 开展补贴活动须**提前7日**公开：补贴资金来源、金额类型方式、起止时间、竞争影响自评报告 |
| | 第8条 | 补贴后须形成实施情况评估报告并向社会公开 |
| 法律风险 | 第9-10条 | 违法违规开展补贴活动的，市场监管部门将依法调查处理 |

**三大外卖平台表态**：美团、淘宝闪购、京东外卖均表态拥护规范要求。

**与反法的关联**：
- 第2条（不得强制经营者承担补贴成本）→ 对应反法第13条第3款（滥用平台规则）；
- 第4条（不得低于成本销售）→ 对应反法第14条（平台强制低价）+第13条第3款；
- 第1条（不得扰乱竞争秩序）→ 对应反法第2条一般条款。

### 7.2.3 平台约谈事件

**（1）北京约谈5家电商平台（2026年6月11日）**

北京市市场监督管理局约谈淘宝（天猫）、京东、拼多多、抖音、小红书，通报第二批典型问题：促销活动虚假宣传、促销规则制定与公示不规范、商品经营者信息未公示、多个平台未公示活动规则。

**监管要求**：
- 将"卷补贴、卷价格"转向"比创新、比服务"；
- 严禁"先涨后降"、虚假折价等价格欺诈行为；
- 严防国补商品骗补套补。

**（2）17家平台联席约谈（2026年5月25日）**

北京市市场监管局联合商务局、文旅局、通管局、税务局、邮政局、版权局、知识产权局、央行北京市分行等9部门，对17家电商平台进行联席约谈，核心要求：**618期间杜绝非理性大额补贴促销活动**。

## 7.3 监管动态与反法诉讼的关联分析

### 7.3.1 行政监管与民事诉讼的衔接

公平竞争审查制度和平台监管措施主要属于**行政监管层面**，但对反不正当竞争民事诉讼具有以下影响：

| 维度 | 影响 |
|---|---|
| 证据效力 | 行政监管部门的约谈记录、整改通知、处罚决定可作为民事诉讼中的有利证据，证明被告行为的违法性 |
| 行为认定 | 监管部门对"内卷式竞争""非理性补贴"的负面定性，可为法院认定违反反法第2条一般条款和第14条提供参考 |
| 赔偿酌定 | 行政处罚金额可作为法院酌定民事赔偿数额的参考因素 |
| 合规抗辩 | 经营者援引合规提示进行整改的记录，可作为"已尽合理注意义务"的抗辩材料 |

### 7.3.2 "内卷式竞争"的法律定性路径

监管层已将平台"拼补贴、拼价格、控流量"定性为"内卷式竞争"，其法律定性路径为：

```
长期大额补贴 → 排除限制竞争（反垄断法）
              → 扰乱竞争秩序（反法第13/14条）
              → 低于成本销售（价格法）
              → 违反商业道德（反法第2条一般条款）
```

### 7.3.3 对诉讼策略的启示

1. **原告视角**：平台经营者受到行政处罚或被约谈后，被侵权方可援引行政文书作为民事诉讼证据，降低举证难度；
2. **被告视角**：平台经营者应及时完善合规体系，留存合规审查记录，以备诉讼中主张"已尽合理注意义务"；
3. **前瞻预判**：外卖补贴十条正式出台后，预计将引发一批行政执法案件，随之产生关联民事诉讼。
